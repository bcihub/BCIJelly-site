8_Reaching MC PMC

Long-term Recordings of Motor and Premotor Cortical Spiking Activity During Reaching in Monkeys

Dataset Overview​

  • Dataset Name: Long-term recordings of motor and premotor cortical spiking activity during reaching in monkeys
  • DOI: 10.48324/dandi.000688/0.250122.1735
  • Authors: Perich, Matthew G.; Miller, Lee E.; Azabou, Mehdi; Dyer, Eva L.
  • Data Types:
    • Neural signals: Multi-channel recordings from M1 / PMd brain regions (192 channels), manually spike sorted or threshold crossings
    • Behavioral signals: Cursor position, velocity, and task-related metadata
  • Experimental Tasks: Center-Out (CO) task and continuous random target task
  • Data Standard: NWB (Neurodata Without Borders)
  • Sampling Rate: 1000 Hz

Processing Goals​

Process the raw NWB data into trial-based samples suitable for neural decoding or regression tasks, including:

  1. Segment spike data by trial;
  2. Resample each trial to a fixed number of time bins (BINNED_LEN = 50);
  3. Construct spike matrices [Time, Channel] for each trial;
  4. Convert each trial's target direction into 8 categorical labels;
  5. Save the processed data as .npz files with features and labels fields.

Parameter Configuration​

ParameterDescriptionExample Value
FSNeural data sampling rate1000 Hz
BINNED_LENNumber of time bins per trial after resampling50
DATA_ROOTPath to raw NWB files./data/sub-C/CO
SAVE_ROOTPath to save processed data./out/sub-C/CO

Data Processing Steps​

1. Iterate Over NWB Files​

  • Only process .nwb files;
  • Extract the file date YYYYMMDD for saving purposes.

2. Open NWB File​

with NWBHDF5IO(nwb_path, 'r') as io:
    f = io.read()

3. Read Trial Information​

  • results: trial success indicator ('R' indicates successful trial)
  • start_time / stop_time: trial start and end times
  • target_dir: trial target direction

Filter successful trials and convert target directions into 8 categories:

tgt_dir_after = tgt_dir_after / np.pi * 4
tgt_dir_after = np.round(tgt_dir_after).astype(np.int32)

4. Read Spike Data​

  • spike_times: spike timestamps for each channel (in seconds)
  • Convert to sample indices: spikes_idx = np.round(spike_times * FS)

5. Trial Segmentation and Spike Matrix Construction​

  • For each trial:
    • Create a zero matrix [trial_len, n_channels];
    • Mark spike times with 1s;
    • Resample evenly to [BINNED_LEN, n_channels].

6. Save Processed Data​

  • Create directory: SAVE_ROOT/YYYYMMDD
  • Save as .npz file:
np.savez(os.path.join(SAVE_ROOT, f'{date}.npz'), features=trials, labels=cly_trials)
  • features: list of [BINNED_LEN, n_channels] matrices
  • labels: categorical target direction for each trial [0, 7]

Visualization Suggestions​

  • Spike raster plots ([time, channel])
  • Trial label distribution
  • Heatmaps of binned spike activity

Notes​

  1. Only successful trials are retained (results == 'R');
  2. BINNED_LEN can be adjusted according to model requirements;
  3. Spike indices should be rounded and bounded to avoid overflow;
  4. Target directions are converted to 8 integer categories for classification models.
