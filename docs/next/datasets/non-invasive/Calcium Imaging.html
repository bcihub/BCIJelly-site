7_Calcium Imaging

Dataset Link​

https://zenodo.org/doi/10.5281/zenodo.10408334

1. Data Acquisition​

Method: Wide-field calcium imaging was used to record fluorescent signals from the mouse cortex, while simultaneously recording behavioral activity. Subjects: 5 mice. Experimental paradigm: Neural activity and behavioral performance were recorded synchronously to analyze the relationship between cortical dynamics and behavior.

2. Data Preprocessing Workflow​

  1. Read experimental configuration information

    Read the mapping between Data_ID and corresponding mouse number (Mouse_#) from data_64_5mice.csv for subsequent output file naming. Read behavioral labels (run/rest segments for each mouse) from runrest_5mice.mat. Obtain the list of .tif filenames corresponding to each mouse number, with each Data_ID representing one imaging experiment.

  2. Image loading and preprocessing

    Use tifffile to read .tif wide-field calcium imaging data, ensuring the data shape is [T, C, H, W] (number of time frames T, number of channels C, spatial resolution H×W). Find the label sequence that best matches the number of image frames (allowing ±5 frame tolerance) and truncate sequences that are too long.

  3. Loading mean / standard deviation

    Load the per-pixel mean and standard deviation matrices for each Data_ID from the precomputed _mean.npy and _std.npy files for normalization.

  4. Frame-wise normalization

    Perform per-pixel normalization for each frame:

    norm=x−meanstd+1×10−5_{\text{norm}} = \frac{x - \text{mean}}{\text{std} + 1\times 10^{-5}}norm​=std+1×10−5x−mean​

    Keep the channel dimension [C, H, W]; if flatten=True, flatten each frame and take the mean across channels to obtain a [C] feature vector.

  5. Sequence windowing

    Set the time window length to T=30; use consecutive 30 frames as one sample, sliding to generate all possible window sequences. The label of each window is determined by majority voting among labels in the window (np.bincount().argmax()).

  6. Naming rules and saving

    The output file naming format is:

    YYYYMMDD_Mouse#.npz

Inside each .npz file:

calcium → windowed image features

label → corresponding window labels

If the file already exists, _v2, _v3, etc., will be appended to the filename to prevent overwriting. All .npz files are saved in the datasets/10408334/data/npz_data/ directory.

def load_image(path):
    img = tifffile.imread(path) 
    if img.ndim == 3:
        img = img[:, np.newaxis, :, :]
    return img.astype(np.float32)
def get_framewise_labels(mat_entry):
    segments = mat_entry[0]  
    label_seq = [seg.flatten() for seg in segments]  
    return np.concatenate(label_seq)
def find_matching_segment(mat_entry, T_img, tolerance=1):
    for seg in mat_entry[0]:
        y = seg.flatten()
        if abs(len(y) - T_img) <= 5:
            if len(y) < T_img:
                y = np.pad(y, (0, T_img - len(y)))
            else:
                y = y[:T_img]
            return y
    raise ValueError(f"Cannot find a label segment that matches the image frame count {T_img}.")

mean_dict = {}
std_dict = {}
save_path = 'datasets/10408334/data/mean_and_std/'

def _unique_path(path):
    if not os.path.exists(path):
        return path
    base, ext = os.path.splitext(path)
    k = 2
    while True:
        cand = f"{base}_v{k}{ext}"
        if not os.path.exists(cand):
            return cand
        k += 1

def build_NTC_direct_and_save(mouse_nums, csv_path, root_npz_dir, T=30, flatten=True):
    os.makedirs(root_npz_dir, exist_ok=True)
    df = pd.read_csv(csv_path)
    id_to_mouse = dict(zip(df['Data_ID'], df['Mouse_#']))

    for mouse_num in mouse_nums:
        for fname in get_filename([mouse_num]): 
            print(f"{fname}.tif")
            X = load_image(os.path.join(image_path, fname + '.tif')) 
            mat_entry = mat_data[f'runrest{mouse_num}']
            Y = find_matching_segment(mat_entry, T_img)
            assert len(X) == len(Y), f"{fname}: Number of image frames {len(X)} != number of labels {len(Y)}"

            mean = mean_dict[fname]  # [H, W]
            std  = std_dict[fname]   # [H, W]

            X_list, Y_list = [], []
            for j in range(len(Y)):
                x = X[j][0]                       # [H, W]
                y = int(Y[j])
                x = (x - mean) / (std + 1e-5)
                x = x[np.newaxis, :, :]          # [C, H, W]
                if flatten:
                    x = x.reshape(x.shape[0], -1).mean(axis=-1)  # [C]
                X_list.append(x)
                Y_list.append(y)

            X_all = np.stack(X_list)              # [T_img, C] or [T_img, C, H, W]
            Y_all = np.array(Y_list)              # [T_img]

            del X_list, Y_list, X, Y
            gc.collect()

            if len(X_all) < T:
                print(f"跳过 {fname}, 帧数 {len(X_all)} < T={T}")
                continue

            N = len(X_all) - T + 1
            X_windows = np.stack([X_all[i:i+T] for i in range(N)])                 # [N, T, ...]
            Y_windows = np.array([np.bincount(Y_all[i:i+T]).argmax() for i in range(N)])  # [N]

            date_part = fname[:8]
            mouse_no  = id_to_mouse.get(fname, f"M{mouse_num}")
            base = f"{date_part}_{mouse_no}.npz"
            save_path = _unique_path(os.path.join(root_npz_dir, base))

            np.savez(save_path, calcium=X_windows, label=Y_windows)

            print(f"{os.path.basename(save_path)} | calcium {X_windows.shape}, label {Y_windows.shape}")
