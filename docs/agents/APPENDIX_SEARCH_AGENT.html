Pseudocode of the AAS search procedure​

Algorithm: Validation-driven architecture search and ensemble selection

Input:
    Training data D_train = (X_train, y_train)
    Validation data D_val = (X_val, y_val)
    Task type task_type in {classification, regression}
    Search configuration:
        max_models_per_round, max_search_rounds, max_depth,
        val_threshold, retain_top_k, ensemble_top_m

Output:
    Best combined model f_best
    Search summary containing retained candidates and the selected ensemble

1.  Discover candidate modules from the local model library.
2.  Initialize the retained candidate set P = empty.

3.  For round r = 1 to max_search_rounds:
4.      Sample up to max_models_per_round candidate architectures under the search constraints.

5.      For each candidate architecture a:
6.          Build a time-series model:
                InputProjection -> searched module blocks -> task head.
7.          Train the model on D_train.
8.          Evaluate the model on D_val.

9.          If the validation metric of a is at least val_threshold:
10.             add a to P.

11.     Keep only the top retain_top_k candidates in P according to validation metric.
12.     If |P| >= retain_top_k:
13.         stop the search.

14. If P is empty:
15.     use the best successfully trained candidates as the fallback pool.

16. Select the top ensemble_top_m candidates from P.
17. Evaluate candidate ensembles on D_val:
        classification: combine class probabilities by mean or stacking;
        regression: average predicted trajectories.
18. Choose the ensemble with the best validation metric.
19. Return the selected ensemble as f_best.

External evaluation:
20. Apply f_best to each held-out test day.
21. Report day-wise metric, average metric, median metric, and minimum metric.

Demo​


import bcijelly as bj
from bcijelly import SearchAgent

loaded = bj.load_data(
    dataset_name="5_Jango_force",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)

agent = SearchAgent(task_type="classification", device="cuda")

agent.fit(loaded.train_dataset, loaded.val_dataset)

by_day = {f"day_{i}": ds for i, ds in enumerate(loaded.test_day_datasets or [])}
results = agent.test(by_day)
print(results)
