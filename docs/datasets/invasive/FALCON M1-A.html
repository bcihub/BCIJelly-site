1_FALCON M1-A

Repository Link：https://snel-repo.github.io/falcon/​

DOI：10.48324/dandi.000941/0.241029.1405​

DANDI ID: 000941 / Version: 0.241029.1405 License: CC-BY-4.0 Access: Open Access

I.Experimental Design and Data Acquisition​

Subjects​

  • Species: Macaca mulatta (Rhesus monkey)
  • Number of subjects: 3 (each dataset split counted as one “subject” in DANDI; actual recordings from one monkey for M1-A, additional monkey data here)

Recording Specifications​

  • Neural Input: Multi-unit activity (MUA) captured from the primary motor cortex (M1) using chronically implanted electrode arrays.
  • Muscle Output: 16-channel intramuscular EMG (iEMG) signals, synchronized with neural data to capture the activation patterns of upper-limb muscles during task execution.

Task Paradigm​

  • Type: Reach-to-grasp, center-out reaching task.
  • Conditions:
    • 4 different objects
    • 8 possible target positions
  • Behavioral events recorded:
    • gocue_time
    • move_onset_time
    • contact_time
    • reward_time
    • result, number, tgt_loc, tgt_obj, obj_id, condition_id

II.Data Processing​

Data Format​

  • Neurodata Without Borders (NWB).
Variable / Field NameDescription
preprocessed_emgiEMG signals from 16 muscles.
eval_maskBoolean mask indicating the trial periods used for evaluation.
unitsSpike times for each recorded unit.
trialsTrial-level metadata, including event times, target/object IDs, and outcomes.

Dataset Structure​

Three data splits provided for the FALCON Challenge:

  • held-in-calib: Many trials; intended for decoder calibration.
  • held-in-minival: Small subset of held-in-calib; for validating model submission format.
  • held-out-calib: Separate sessions with fewer trials; for few-shot recalibration.

Data Processing Pipeline​

This pipeline is designed to convert raw neural recordings into standardized tensor formats suitable for model training, covering the entire workflow from data ingestion to feature and label construction.

1. Data Loading

  • Utilize the pynwb library or the dandi Python client to directly read and load raw .nwb files from either local storage or cloud repositories.

2. Signal Preprocessing & Binning

  • Precisely align the extracted neural spike sequences with specific behavioral event timestamps. Subsequently, perform fixed-step binning operations on the continuous timeline to discretize the data.

3. Feature Extraction

  • Extract core neurodynamic features from the aligned and binned data. Specifically, calculate the Mean Firing Rate and construct Peri-Stimulus Time Histograms (PSTH) to capture trial-averaged neural responses.

4. Label Generation & Windowing

  • Map individual trials to their corresponding discrete experimental conditions to generate supervised labels. Simultaneously, rigorously define extraction time windows relative to key behavioral events to ensure temporal consistency across samples.

III.Demo​

import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="1_FALCON M1-A",            # Select the dataset
    protocol="single_day",                  # "single_day" | "cross_day"; demonstrates single-day decoding here
    subject_index=0,                        # Range: [0, n_subjects - 1]
    day_index=0,                            # Range: [0, n_days - 1]
    seed=42,                                # int; random seed for splitting test and validation sets in single-day decoding
    single_day_test_size=0.2,               # Range: (0, 1); proportion of the single-day data allocated to the test set
    single_day_val_ratio=0.125,             # Range: [0, 1); only valid when val_dataset is not None; represents the ratio of val within the train+val split
    # auto_download_missing=True,           # Automatically download from Hugging Face if local data is missing; defaults to True
    verbose=False,
)

train_dataset, val_dataset, test_dataset = loaded_data.train_dataset, loaded_data.val_dataset, loaded_data.test_dataset

train_neural_signal, train_label = train_dataset
val_neural_signal, val_label = val_dataset
test_neural_signal, test_label = test_dataset

print(train_neural_signal.shape, train_label.shape)
print(val_neural_signal.shape, val_label.shape)
print(test_neural_signal.shape, test_label.shape)
