2_Transformer

I. Model Overview​

This model provides a Transformer sequence decoding architecture specifically designed for neurophysiological signals (such as Spikes, LFPs, etc.). Brain-Computer Interface (BCI) data typically exhibits complex temporal dependencies and cross-channel correlations. Traditional RNN models are prone to the vanishing gradient problem when processing long sequences. This model breaks through this limitation. Its core hypothesis is that utilizing the Self-Attention mechanism can directly establish global dependencies between arbitrary time steps, thereby more efficiently capturing the dynamic representations of neural population activities.

From the perspectives of code implementation and network topology, this Transformer architecture consists of the following core modules:

  1. Input Projection

    The raw input typically consists of neural signals with high-dimensional feature channels. The model first maps the feature dimensions to a unified embedding dimension via a linear projection layer, ensuring feature space consistency for subsequent attention computations.

  2. Positional Encoding

    For tasks with strong temporal dependencies, such as continuous kinematic trajectory decoding, the model introduces fixed-period Sine/Cosine Positional Encoding. This mechanism injects absolute temporal information into the inherently non-directional self-attention mechanism, enabling the model to perceive the chronological sequence of the spike trains.

  3. Multi-Head Attention Encoder

    The core backbone network is constructed by stacking multiple Transformer encoder layers. Through the multi-head mechanism, the model computes the scaled dot-product attention of Queries, Keys, and Values in parallel across different subspaces. This mechanism completely shatters the constraints of temporal distance, enabling the model to directly establish neural action potential dependencies spanning hundreds of milliseconds with a global receptive field.

  4. Dual-Track Task Head

    To accommodate generalized BCI paradigms, the model incorporates highly flexible decoding logic at the output end:

    • Pooled Decoding: For discrete intention recognition or entire-signal classification, the model performs average pooling across the temporal dimension, subsequently connecting to a linear classification head.
    • Seq-Target Decoding: When sequence target prediction is enabled, the model bypasses the pooling layer and directly applies a linear projection to the hidden states at each time step. This outputs a feature tensor containing the temporal dimension, precisely adapting to the real-time regression prediction of continuous kinematic coordinates (e.g., 2D spatial coordinates).

IV. Model Debugging & Evaluation​

To accurately measure the model's performance in neural decoding tasks, the system features a built-in dual-track optimization strategy and a comprehensive evaluation matrix:

  • Task-Isolated Optimization

    For classification tasks, the model employs a standard Adam optimizer paired with Cross-Entropy Loss. Conversely, for complex continuous regression tasks, the model switches to the more robust AdamW optimizer, mandating the integration of a Cosine Annealing learning rate scheduler and Smooth L1 Loss. This ensures convergence smoothness while tolerating extreme outliers in neural signals.

  • Multi-Dimensional Metrics

    • Classification: Each validation step not only outputs accuracy and cross-entropy loss but also comprehensively records the confusion matrix, facilitating the tracking of biases caused by class imbalance.
    • Regression: Discarding the reliance on a single Mean Squared Error metric, the model synchronously calculates the Mean Squared Error, Mean Absolute Error, and the Coefficient of Determination. The Coefficient of Determination provides an intuitive reflection of the proportion of neural dynamic variance explained by the model.

  • Automated Logging & Visualization

    The system tracks training and validation metrics as well as learning rate changes in real-time for each epoch. The platform can generate dual-Y-axis training trend curves (supporting side-by-side comparisons of loss descent and metric ascent) and optimal metric bar charts for batch tasks with a single click.

V. Application Scenarios​

Based on engineering practices, this Transformer architecture has been designed to adeptly handle classic invasive and non-invasive BCI datasets at the level of top-tier conferences and journals, demonstrating exceptional domain transferability:

  • Kinematic Regression

    Targeted at non-human primate BCI datasets. By enabling sequence target prediction and strictly aligning the spatial coordinate dimensions, the model can accurately read cortical Spike signals to reconstruct the continuous kinematic coordinates of monkeys performing tasks frame by frame.

  • Discrete State Classification

    For datasets involving fine finger movement recognition, music, or emotional state classification, the model utilizes the global pooling mechanism to perform dimensionality reduction and feature fusion on entire high-dimensional inputs, effectively outputting multi-class intention classification results.

  • Visual Coding

    When processing probe data, the attention distribution mechanism of the Transformer allows researchers to reverse-track long-term sequence data, uncovering non-linear dependency features between stimulus signals and the firing activities of neuronal populations.

  • Offline Global Analysis & Block-Online Decoding

    Unlike the frame-by-frame progression of RNNs, the Transformer's reliance on global context windows endows it with unparalleled long-term feature mining capabilities. This gives it a dominant advantage in Offline neural decoding, sleep staging analysis, or cognitive state reconstruction. For BCI control tasks requiring real-time responses, the Transformer is better suited for a Block-Online paradigm—introducing an extremely brief computational latency (e.g., accumulating short data blocks) in exchange for exceptionally high global resolution of spatial and temporal neural correlations.
