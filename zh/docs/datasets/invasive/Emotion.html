13_Emotion

数据集链接：https://neurotycho.org/emotional-movie-task​

一、实验设计与数据采集​

被试对象及数量​

  • 物种：恒河猴
  • 被试者数量：多只

记录范式​

  • 记录类型：猴子执行任务期间，同时记录脑电图（ECoG）数据和眼动追踪数据。未进行运动追踪。

任务范式​

  • 任务类型：被动观看任务（Passive Viewing Task）。
  • 刺激材料 (Stimuli)：6 段不同的电影片段（Movie clips）。
  • 被试者状态 (Subject State)：被试者（猴子）处于静坐状态，且头部被固定（Head fixed），以减少运动伪影并确保视觉刺激的稳定性。
  • 反馈与奖惩机制 (Feedback & Reward)：无食物奖励（No food reward）。被试者不需要执行特定的主动交互动作来获取奖励，这排除了操作性条件反射的影响。
  • 同步数据采集 (Synchronized Recording)：
  • 神经记录：皮层脑电（ECoG），采样率为 1KHz。
  • 行为记录：眼动追踪（Eye-tracking），采样率为 120Hz。
  • 同步与约束：两种信号的起止时间严格硬件/软件同步；实验明确排除了肢体运动追踪（No motion tracking），将分析焦点集中在视觉输入、眼动行为与皮层神经活动的映射上。

采样率​

  • 脑电图：1kHz
  • 眼动追踪数据：120Hz

通道配置​

  • 本流程用于处理 128 通道 ECoG LFP 数据，并将其转换为可供后续分类/模式识别使用的 binned spike 表示。

二、使用示例​

1.分类任务​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="13_Emotion",        
    protocol="single_day",      # single_day | cross_day | ...
    task="classification",
    subject_index=0, 
    day_index=0,                # cross_day时参数无效
    split_ratio="7:1:2",        
    verbose=False,
)

train_dataset, val_dataset, test_dataset = loaded_data.train_dataset, loaded_data.val_dataset, loaded_data.test_dataset

train_neural_signal, train_label = train_dataset
val_neural_signal, val_label = val_dataset
test_neural_signal, test_label = test_dataset

print(train_neural_signal.shape, train_label.shape)
print(val_neural_signal.shape, val_label.shape)
print(test_neural_signal.shape, test_label.shape)

三、注意事项​

  • 滤波器频率设置：低截止 > 0 Hz，高截止 < Nyquist 频率
  • trial 截断：如果某 trial 末尾不足 trial_duration_sec 秒，会被丢弃
  • bin_size 需整除 target_timesteps，否则会丢弃最后不完整的 bin
  • 阈值检测灵敏度：可调整系数 2σ 为 1.5σ 或 3σ
  • 不应期单位：这里的 refractory_ms 在降采样后依然按“步”计算，需注意换算
