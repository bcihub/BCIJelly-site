8_Reaching MC PMC

Long-term recordings of motor and premotor cortical spiking activity during reaching in monkeys

数据集链接：https://doi.org/10.48324/dandi.000688/0.250122.1735​

DOI：10.48324/dandi.000688/0.250122.1735​

一、实验设计与数据采集​

被试对象及数量​

  • 物种：恒河猴
  • 被试者数量：4

记录位置​

  • 植入配置：实验涉及四只恒河猴，均在初级运动皮层（M1）或背侧前运动皮层(PMd)慢性植入高通量电极阵列。
  • 硬件通道： 采用 192 通道电极阵列进行神经活动探测。
  • 处理模式： 记录包含原始电位下的多单位活动 (MUA)，并提供手工脉冲排序 (Manual Spike Sorting) 或阈值穿越 (Threshold Crossing) 两种特征提取模式。
  • 运动学数据： 同步实时记录被试者操控光标的二维位置 (Position) 与速度 (Velocity) 矢量。
  • 任务逻辑： 集成任务触发的时间点、试次 (Trial) 成功状态及相关任务属性等关键元数据。

任务范式​

  • 实验任务：中心-外任务（Center-Out, CO）和连续随机目标任务

采样率​

  • 1000 Hz

二、数据处理​

数据格式​

  • Neurodata Without Borders（NWB）
参数说明示例值
FS神经数据采样率1000 Hz
BINNED_LEN每个 trial 重采样后的时间步数50
DATA_ROOT原始 NWB 文件路径./data/sub-C/CO
SAVE_ROOT处理后数据保存路径./out/sub-C/CO

数据处理流程​

本流程旨在将原始的 .nwb 数据处理为适合神经解码或回归任务的标准化试次 (trial) 样本集。

1. 文件遍历与数据读取 (File Traversal & Data Loading)

  • 遍历目标目录，专门筛选并处理 .nwb 格式文件，同时提取文件中的日期信息 (YYYYMMDD) 作为后续存储的文件名标识。随后，系统调用 pynwb.NWBHDF5IO 接口以只读模式 ('r') 安全打开并加载 NWB 文件的底层数据。

2. 试次筛选与标签映射 (Trial Filtering & Label Mapping)

  • 从读取的数据中提取核心试次信息，包括起止时间 (start_time / stop_time)、目标方向 (target_dir) 以及执行结果 (results)。系统严格筛选出标记为成功（results == 'R'）的试次，并通过数学变换（如 tgt_dir_after / np.pi * 4 并取整操作），将原始连续的目标方向精确映射为 8 个离散的类别标签（取值范围 [0, 7]）。

3. 脉冲转换与特征矩阵重采样 (Spike Conversion & Resampling)

  • 读取各神经通道的脉冲时间点 (spike_times，单位为秒)，将其乘以系统采样率 (FS) 转换为离散的采样索引。针对每个切分出的试次，系统首先初始化一个形状为 [trial_len, n_channels] 的零矩阵，并将对应索引的脉冲位置标记为 1；随后，对该稀疏矩阵执行等宽重采样，将其强制对齐至固定的时间步数（设定 BINNED_LEN = 50），最终构建出形状为 [BINNED_LEN, n_channels] 的特征矩阵。

4. 结果导出与质检可视化 (Data Export & Visualization)

  • 系统在 SAVE_ROOT/YYYYMMDD 路径下自动创建目录，并将构建好的样本列表压缩保存为 {date}.npz 文件。为确保数据质量，流程强烈建议配套生成三类可视化图表以供复核：
    • 脉冲光栅图 (Spike raster plot, [time, channel])
    • 试次标签分布图 (Trial label distribution)
    • 重采样后的脉冲热力图 (Spike heatmap)。

三、使用示例​

1. 数据集加载与检查​

import bcijelly as bj
loaded_data = bj.load_data(
    dataset_name="8_Reaching MC PMC",            # 选择数据集
    protocol="single_day",                  # "single_day" | "cross_day" | ...
    task="classification",                  # "classification" | "regression" | ...
    subject_index=0,                        # 范围: [0, n_subjects - 1]
    day_index=0,                            # 范围: [0, n_days - 1]
    split_ratio="7:1:2",                    # 表示训练(-验证)-测试集划分比例，格式为"m:n"或"m:v:n"
    verbose=False,
)
train_dataset, val_dataset, test_dataset = loaded_data.train_dataset, loaded_data.val_dataset, loaded_data.test_dataset
train_neural_signal, train_label = train_dataset
val_neural_signal, val_label = val_dataset
test_neural_signal, test_label = test_dataset

print(train_neural_signal.shape, train_label.shape)
print(val_neural_signal.shape, val_label.shape)
print(test_neural_signal.shape, test_label.shape)

2. 全流程训练与评估​

import bcijelly
# 加载模型与数据集
import bcijelly.model.transformer.classifier as module
loaded_data = bcijelly.load_data(
    "8_Reaching MC PMC",
    protocol="single_day", task="classification",
    subject_index=0, day_index=0, split_ratio="7:1:2",
)
# 执行训练、验证与测试，生成结果汇总
train_result = module.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = module.test(loaded_data.test_dataset)
bcijelly.summary(
    loaded_data, module, train_result, test_result,
    mode="compact",
)

四、注意事项​

  1. trial 仅保留成功实验（results == 'R'）
  2. BINNED_LEN 可根据模型需求调整
  3. spike 时间索引需四舍五入并确保不越界
  4. target_dir 转为 8 类整数便于分类模型输入
