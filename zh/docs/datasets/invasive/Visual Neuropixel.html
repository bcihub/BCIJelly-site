9_Visual Coding Neuropixels

Allen Brain Observatory — Visual Coding Neuropixels 数据集（NWB，通过 AllenSDK 下载）。该数据集包含 58 个实验、每次实验最多 6 根 Neuropixels 探针，覆盖皮质、丘脑、海马等区域；用于系统性研究小鼠视觉系统在被动刺激条件下的放电活动。本流程以 drifting gratings（8 个方向）刺激为例，演示完整的会话筛选、单元选择、脉冲矩阵构建、时间 binning、与数据保存。关于 drifting gratings 的 8 个方向与频率设置可参考官方说明。

数据集链接：https://portal.brain-map.org/circuits-behavior/visual-coding-neuropixels​

一、实验设计与数据采集​

被试对象及数量​

  • 物种：小鼠
  • 被试者数量：多只

记录范式​

  • 记录位置：覆盖皮层、丘脑、海马等区域

任务范式​

  • 标准化刺激流程：每只被试小鼠均接被一套标准化的视觉刺激方案，以确保实验在时间维度上的可重复性，并支持跨脑区视觉响应的并行映射 (Parallel mapping)。
  • 经典刺激特征刻画：
  • 刺激内容：包含 Gabor 斑块 (Gabor patches)、全视野漂移光栅 (Full-field drifting gratings) 及移动点 (Moving dots)。
  • 实验目的：旨在精确表征单个神经元的调谐特性 (Tuning properties)，如方向选择性或空间频率偏好。
  • 自然/生态相关性实验：
  • 刺激内容：呈现一系列复杂的自然图像 (Natural images) 与视频片段 (Natural movies)。
  • 实验目的：在更符合生态相关性 (Ecologically relevant) 的真实场景下，探索不同视觉皮层区域间的协同交互与动力学机制。

二、使用示例​

1.分类任务​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="9_Visual Coding Neuropixels",        
    protocol="single_day",      # single_day | cross_day | ...
    task="classification",
    subject_index=0, 
    day_index=0,                # cross_day时参数无效
    split_ratio="7:1:2",        
    verbose=False,
)

train_dataset, val_dataset, test_dataset = loaded_data.train_dataset, loaded_data.val_dataset, loaded_data.test_dataset

train_neural_signal, train_label = train_dataset
val_neural_signal, val_label = val_dataset
test_neural_signal, test_label = test_dataset

print(train_neural_signal.shape, train_label.shape)
print(val_neural_signal.shape, val_label.shape)
print(test_neural_signal.shape, test_label.shape)

三、注意事项​

  • 找不到 drifting_gratings：并非每个会话都含该刺激类型，脚本已在缺失时跳过；可考虑支持 static_gratings 或自然影片等刺激类型。
  • 时间步与 bin 大小：expect_num_bins 越大时间分辨率越高，但输入张量更大；可依据下游模型/显存调整。
  • 单位质量控制：当前仅以 SNR ≥ 2 过滤；可结合 firing_rate、ISI violations 等 AllenSDK 预计算指标进一步筛选。
