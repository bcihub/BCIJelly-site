5_SEED VII

数据集链接：https://bcmi.sjtu.edu.cn/~seed/seed-vii.html​

DOI：10.1109/TAFFC.2024.3485057​

一、实验设计与数据采集​

被试对象及数量​

  • 人数：20 名参与者（10 男，10 女），年龄 19--26 岁（平均 22.5岁，标准差 1.8）。
  • 条件：全部为右利手，具有正常或矫正至正常的视力及正常听力。
  • 筛选：通过艾森克人格问卷（EPQ）筛选。

记录范式​

  • 脑电（EEG）：使用 Neuroscan 设备采集 62 通道 EEG 信号。

  • 眼动（Eye-tracking）：使用 Tobii Pro Fusion 眼动仪同步采集眼动参数。

任务范式​

  • 情绪诱发基于 80 段视频（包含快乐、悲伤、恐惧、愤怒、厌恶、惊讶各 12 段，以及中性 8 段）。每段视频时长 2 至 5 分钟。

  • 每位受试者完成 4 个会话（sessions）（分不同日期进行）。

  • 每个会话包含 20 个试次（trials）。每个会话仅诱发 7 种情绪中的 5 种，以避免频繁切换情绪状态。

  • 每个试次分为两部分：

    • 观看诱发视频（视频前后有 3 秒倒计时提示）。

    • 自我情绪评估：在会话结束时，受试者回顾视频并给出一个连续的评分（0-1 之间）。1 代表目标情绪被极好地诱发，0 代表未诱发。对于中性视频，情绪波动评 0 分，保持自然状态评 1 分。

采样率​

  • 原始 EEG 信号采样率为 1000 Hz，在特征提取前被下采样至 200 Hz。

二、数据处理​

数据格式​

  • 原始脑电数据：以 .cnt 格式存储。

  • 原始眼动数据：以 .tsv 格式存储。

  • 特征数据：以 MATLAB 的 .mat 格式存储。

  • 辅助信息：以 .csv 和 .xlsx 格式存储。

  • 处理后用于 Python 的重组数据：通常打包为 .npz 格式供深度学习模型调用。

数据处理流程​

本流程旨在将原始脑电（EEG）记录数据转换为适用于模型训练的标准化张量格式，涵盖了从数据加载、信号预处理、特征提取到标签对齐的完整流水线。

1. 数据加载与解析 (Data Loading)

  • 通过特定的 load_data() 函数，按会话 ID (session_id) 和被试 ID (sub_id) 遍历指定目录，直接读取并加载以 .npz 格式重组存储的数据文件。

2. 信号预处理与切片 (Signal Preprocessing & Segmentation)

  • 首先使用 MNE-Python 工具对原始脑电信号进行可视化坏道检查与插值修复。接着应用 0.1 Hz - 70 Hz 的带通滤波器以及 50 Hz 的陷波滤波器，以滤除低频噪声和电源工频干扰，并将采样率从 1000 Hz 统一降频至 200 Hz。随后，将连续信号严格切分为 4 秒且互不重叠的时间片段作为独立样本。

3. 特征提取与平滑 (Feature Extraction & Smoothing)

  • 从切分好的每个 4 秒片段中，提取 5 个经典频段（δ、θ、α、β、γ）的差分熵（DE）特征。随后，采用线性动态系统（LDS）或滑动平均滤波方法对特征进行平滑处理以进一步降噪，最终构建出形状为 [样本数, 频带数(5), 通道数(62)] 的三维特征张量（features）。

4. 标签生成与时间同步 (Label Generation & Synchronization)

  • 提取每个视频对应的目标情绪类别（7 类）以及受试者的主观评分（0-1）作为标签来源。严格按照视频播放的时间窗对齐特征与标签，确保每一个 4 秒的片段都有对应的情绪真值，最终生成对齐的一维标签数组（labels）。

三、使用示例​

import bcijelly as bj
loaded_data = bj.load_data(
    dataset_name="5_SEED VII",            # 选择数据集
    protocol="single_day",                  # "single_day" | "cross_day" | ...
    task="classification",                  # "classification" | "regression" | ...
    subject_index=0,                        # 范围: [0, n_subjects - 1]
    day_index=0,                            # 范围: [0, n_days - 1]
    split_ratio="7:1:2",                    # 表示训练(-验证)-测试集划分比例，格式为"m:n"或"m:v:n"
    verbose=False,
)
train_dataset, val_dataset, test_dataset = loaded_data.train_dataset, loaded_data.val_dataset, loaded_data.test_dataset
train_neural_signal, train_label = train_dataset
val_neural_signal, val_label = val_dataset
test_neural_signal, test_label = test_dataset

print(train_neural_signal.shape, train_label.shape)
print(val_neural_signal.shape, val_label.shape)
print(test_neural_signal.shape, test_label.shape)

2. 全流程训练与评估​

import bcijelly
# 加载模型与数据集
import bcijelly.model.transformer.classifier as module
loaded_data = bcijelly.load_data(
    "5_SEED VII",
    protocol="single_day", task="classification",
    subject_index=0, day_index=0, split_ratio="7:1:2",
)
# 执行训练、验证与测试，生成结果汇总
train_result = module.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = module.test(loaded_data.test_dataset)
bcijelly.summary(
    loaded_data, module, train_result, test_result,
    mode="compact",
)
