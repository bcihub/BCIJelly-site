2_transformer

一、模型简介​

本模型提供了一套专门针对神经生理信号（如 Spike、LFP 等）设计的 Transformer 序列解码架构。脑机接口数据通常具有复杂的时序依赖性和跨通道相关性，传统 RNN 模型在处理长序列时易面临梯度弥散问题。本模型突破了这一局限，其核心假设在于：利用自注意力机制（Self-Attention）可以直接建立任意时间步之间的全局依赖关系，从而更高效地捕获神经群体活动的动态表征。

从代码实现与网络拓扑来看，本 Transformer 架构由以下核心模块构成：

  1. 输入投影层 (Input Projection)

    原始输入通常为高维特征通道的神经信号。模型首先通过线性投影层将特征维度映射至统一的嵌入维度，确保后续注意力计算的特征空间一致性。

  2. 正弦位置编码 (Positional Encoding)

    针对连续运动轨迹解码等强时序依赖任务，模型引入了固定周期的正弦/余弦位置编码。此机制将绝对时序信息注入到无向的自注意力机制中，使模型能够感知脉冲序列的发生顺序。

  3. 多头自注意力编码器 (Multi-Head Attention Encoder)

    核心骨干网络由多层 TransformerEncoderLayer 堆叠而成。通过多头机制，模型在不同的子空间中并行计算查询（Query）、键（Key）和值（Value）的缩放点积注意力。该机制彻底打破了时序距离的限制，使得模型能够以全局感受野直接建立跨越数百毫秒的神经动作电位依赖关系。

  4. 双轨任务解码头 (Dual-Track Task Head)

    为适配泛用的 BCI 范式，模型在输出端设计了高度灵活的解码逻辑：

    • 全局池化解码 (Pooled Decoding)： 对于离散的意图识别或整段信号分类，模型在时间维度上进行均值池化，随后连接线性分类头。
    • 逐帧时序解码 (Seq-Target Decoding)： 当开启序列目标预测时，模型绕过池化层，直接对每个时间步的隐藏状态进行线性映射，输出包含时间维度的特征张量，精准适配连续运动学坐标（如二维空间坐标）的实时回归预测。

二、数据预处理与特征工程​

为了保障 BCIJelly 平台在处理不同来源、不同范式侵入式数据集时的鲁棒性，本模型的底层数据管道内置了严格的张量化规范与预处理逻辑：

  • 多键值兼容的特征张量化 (Multi-key Tensorization)

    模型原生支持基于高维数据解析。数据流自动向下兼容多种神经科学社区常用的特征键与标签键，并强制验证输入张量为标准的三维格式（样本数、时间步长、特征通道）。

  • 自适应标签编码 (Adaptive Label Encoding)

    针对分类任务中常见的异构标签（如 One-hot 编码、字符串、浮点数下标），系统内置了自动化清洗与解析函数。该机制可自动清洗乱码、解析字节串，并将多类别标签映射为从 0 开始的稠密整数 ID，确保与交叉熵损失函数的无缝衔接。

  • 回归目标的统计学归一化 (Statistical Normalization)

    对于连续回归任务，模型不仅要求目标张量在维度上严格对齐，更在训练前强制执行统计学拟合。系统会自动计算训练集分布中特征与标签的均值及标准差，并在验证与测试阶段进行严格的 Z-score 标准化与逆标准化转换，以此保障梯度下降的稳定性。

三、参数说明​

1. 顶层数据与调度参数 (Top-Level Data & Scheduling)​

此模块负责定义实验的宏观环境、数据路径以及模型训练的基本调度属性。

参数 / CLI Flag类型适用任务默认值详细说明与覆写建议 (Override)
task_typestr通用"cls"核心分发器：定义当前任务类型。可选 "cls"(分类) 或 "reg"(回归)。
domain_modestr通用"single-day"场景定义：决定当前数据是否存在严重分布偏移。可选 "cross-day"(跨天)，开启后将自动触发强正则化与早停策略。
train_datapathstr通用None训练集文件路径。如无独立验证集，将自动切分。
test_datapathstr通用None测试集文件路径。用于在缓存中直接调用测试评估。
batch_sizeint通用32训练时的批处理大小。
epochsint通用100完整遍历训练数据集的最大轮数。[核心策略]：cross-day 场景极易过拟合，需削减至 60；若为 Jango_force 跨天数据集，则覆写为 90。
lrfloat通用0.0001优化器的初始学习率。
patienceint通用20基于验证集指标的 Early Stopping 耐心轮数。[核心策略]：cross-day 场景建议收紧至 15；Jango_force 数据集维持 20。
val_ratiofloat通用0.2内部自动划分验证集的比例。分类任务会自动启用分层抽样以保障类别均衡。
zscore_normbool通用True是否在数据输入模型前，强制进行特征维度的 Z-score 标准化。
seedint通用42全局随机种子，必须固定以保证多组对比实验的完全可复现性。
devicestr通用"auto"硬件调度。优先寻找 CUDA，否则自动回退到 CPU 运算。

2. Transformer 核心架构参数​

本模块不仅定义了 Transformer 的网络拓扑，还包含了针对不同数据分布（特别是跨天解码）的专属正则化策略。

参数名称类型默认值详细说明 (含高阶调参建议)
num_layersint2编码器层的堆叠层数，控制网络的深度与抽象能力。注：针对 Jango_force 跨天分类任务，建议提升至 3。
d_modelint64核心特征嵌入维度。原始输入通道将通过线性投影被映射至此统一维度。注：针对 Jango_force 跨天分类任务，建议翻倍至 128。
n_headsint4多头注意力机制的“头”数。必须满足嵌入维度能被其整除。注：针对 Jango_force 跨天分类任务，建议提升至 8。
dropoutfloat0.1注意力权重与前馈网络中的神经元丢弃概率，用于抑制噪声。[核心策略]：若进行 cross-day 跨天解码，需提升至 0.3；Jango_force 数据集设为 0.2。
weight_decayfloat0.0优化器的 L2 正则化权重。[核心策略]：单天任务设为 0；若是 cross-day 跨天任务，需引入惩罚项，建议覆写为 0.001（Jango_force 数据集设为 0.0001）。
seq_targetboolFalse序列目标开关。开启表示执行时间步到时间步的逐帧预测；关闭表示池化后输出单一预测。

五、模型调试与评估​

为了精准衡量模型在神经解码任务中的表现，系统内置了双轨制的优化策略与综合评估矩阵：

  • 任务隔离的优化器调度 (Task-Isolated Optimization)

    在分类任务中，模型采用标准的 Adam 优化器配合 CrossEntropyLoss；而在复杂的连续回归任务中，模型切换为更具鲁棒性的 AdamW 优化器，并强制绑定余弦退火学习率调度器与平滑 L1 损失，以此在容忍神经信号极端异常值的同时，保证收敛的平滑度。

  • 多维评估指标 (Multi-Dimensional Metrics)

    • 分类：每次验证不仅输出准确率与交叉熵损失，更全面记录混淆矩阵，便于追踪类别不平衡导致的偏置。
    • 回归：抛弃单一的均方误差，模型同步计算均方误差、平均绝对误差和决定系数。决定系数能够直观反映模型对神经动力学方差的解释比例。

  • 自动化日志与可视化 (Automated Logging & Visualization)

    系统实时追踪每个轮次的训练与验证指标以及学习率变化。平台可一键生成双 Y 轴的训练趋势曲线（支持并排对比损失下降与指标攀升态势）以及批量任务的最优指标柱状图。

六、应用说明​

依据工程实践，本 Transformer 架构已被设计为足以应对顶会/顶刊级别的经典侵入式与非侵入式 BCI 数据集，具备极高的领域迁移能力：

  • 连续运动学解码 (Kinematic Regression)

    针对非人灵长类脑机接口数据集。通过开启序列目标预测并严格对齐空间坐标维度，模型能够精确读取皮层 Spike 信号，逐帧重构出猴子在执行任务时的连续运动学坐标。

  • 离散意图与状态识别 (Discrete State Classification)

    对于手指精细运动识别、音乐或情感状态分类等数据集，模型利用全局池化机制，对整段高维输入进行降维和特征融合，有效输出多分类的意图判定结果。

  • 视觉皮层编码分析 (Visual Coding)

    在处理探针数据时，Transformer 的注意力分布机制允许研究人员逆向追踪长时序数据，挖掘刺激信号与神经元群发放活动之间的非线性依赖特征。

  • 离线全局分析与块级在线解码 (Offline Global Analysis & Block-Online Decoding)

    与 RNN 的逐帧步进不同，Transformer 对全局上下文窗口的依赖赋予了其无与伦比的长时程特征挖掘能力，这使其在离线（Offline）神经解码、睡眠分期分析或认知状态重构中表现出统治级优势。对于需要实时响应的 BCI 控制任务，Transformer 更适合采用块级在线（Block-Online）范式——即引入极短暂的计算延迟（如积攒短暂的数据块），以此换取对空间与时间神经相关性的极高全局分辨率。
