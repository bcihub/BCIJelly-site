3_EEGMMI

数据预处理流程说明

所用数据集链接：​

https://physionet.org/content/eegmmidb/1.0.0/

1. 项目概述​

本项目提供了一种预处理EEG Motor Movement/Imagery数据集的方式，将原始的EEG信号转化为用于模型训练的信号。项目只使用了EEG Motor Movement/Imagery数据集的两个子集。第一个子集仅包含执行的左手或右手运动任务。第二个子集仅包含想象的左手或右手运动任务。在这两个子集中，109名受试者中共使用了103名，由于数据注释错误，第38、88、89、92、100和104名受试者被省略。

2. 预处理过程​

每次试验的输入数据被切分为 (64, W) 个维度，其中 W 表示时间维度，64 表示通道数。由于执行和想象任务的每次试验包含 4 到 4.1 秒的连续、持续的执行或想象运动，因此所有试验都被截断为恰好包含 4 秒的数据，以 160 Hz 的频率采样，总共 640 个样本，以保持一致的数据集。

使用滑动窗口方法，每次试验中的 640 个样本被切分成 8 个互不重叠的窗口，每个窗口包含 80 个样本。每个窗口都赋予与原始试验相同的目标标签。有效地将数据集的大小增加了 8 倍，提供了更多可用的数据。

使用 Gumpy BCI 库中的信号处理模块 (https://www.mdpi.com/2073-431X/9/3/72#B21-computers-09-00072) ] 处理脑电信号。对数据应用陷波滤波器，以消除 60 Hz 频率的交流电 (AC) 噪声。进一步对数据进行 2 Hz 至 60 Hz 范围内的带通滤波，阶数为 5。

3. 核心函数​

3.1 load_data（）函数​

load_data(nr_of_subj,trial_type,chunk_data,chunks,base_folder,processed_out_folder,sample_rate,samples,cpu_format,preprocessing,hp_freq,bp_low,bp_high,notch,hp_filter,bp_filter,artifact_removal)

  • 功能： 加载并预处理多受试者的脑电（EEG ）实验数据，支持带通/高通/陷波滤波、伪迹去除和数据切片。返回预处理后的特征矩阵和标签数组，同时将预处理后的数据以npz格式（在4.4部分具体介绍）保存到processed_out_folder指定的文件夹中。
  • 输入参数说明：
    • nr_of_subj：int = 109，加载的受试者数量。
    • trial_type：int = 1，试验类型标识（如 "执行运动类型为1", "想象运动类型为2", "执行运动和想象运动两种类型为3" ）。`
    • chunk_data：bool = True，是否将每个 trial 切分为多个时间片段（chunk）。
    • chunks：int = 8，每个 trial 切分的片段数，仅当 chunk_data=True 时生效。
    • chunk_data：bool = True，是否将每个 trial 切分为多个时间片段（chunk）。
    • cpu_format：bool = True，是否将输出数据格式化为 CPU 可处理的 Numpy 数组（否则可能是 GPU Tensor）。
    • preprocessing：bool = True，是否执行预处理（滤波、伪迹去除等）。
    • hp_freq：float= 0.5，高通滤波器截止频率（Hz），仅在 hp_filter=True 时生效。
    • bp_low：float= 2.0，带通滤波低截止频率（Hz），仅在 bp_filter=True 时生效。
    • bp_high：float= 60.0，带通滤波高截止频率（Hz），仅在 bp_filter=True 时生效。
    • notch：bool = True，是否启用陷波滤波（用于去除工频干扰，如 50Hz/60Hz）。
    • hp_filter：bool = False， 是否执行高通滤波（覆盖hp_freq` 设置）。
    • bp_filter：bool = True，是否执行带通滤波（覆盖 bp_low 和 bp_high 设置）。
    • artifact_removal：bool = True，是否进行伪迹去除。
    • base_folder：str= ""，数据集所在文件夹名称。
    • processed_out_folder：str= ""，预处理完成的数据集所在文件夹名称。
    • sample_rate：int= 160，采样率。
    • samples：int= 640，采样点。
  • 输出参数说明：
    • all_features：预处理后的脑电信号特征数据，形状通常为 (样本数, 通道数, 时间点数)
    • all_labels：对应样本的标签（样本数，标签）,标签为运动的是左手还是右手

4. 用法​

4.1 准备数据​

将包含所有数据的文件夹放到 ‘‘base_folder’’ 中

4.2 运行预处理函数 “load_data()”​

python load_data(
    nr_of_subj=109,
    trial_type=1,
    chunk_data=True,
    chunks=8,
    base_folder= "./data/",
    processed_out_folder="./out_data/",
    sample_rate=160,
    samples=640,
    cpu_format=False,
    preprocessing= rue,
    hp_freq=0.5,
    bp_low=2.0,
    bp_high=60.0,
    notch=True,
    hp_filter=False,
    bp_filter=True,
    artifact_removal=True
)

设置 "trial_type" 值，可以得到三种类型的预处理数据，另外，可以根据自己设置的参数得到不同的数据预处理结果。

4.3 结果预览​

├── out_data/
│   ├── EMM_task1_preprocessed.npz           # 执行运动类型的预处理数据
│   ├── EMM_task2_preprocessed.npz           # 想象运动类型的预处理数据
│   ├── EMM_task3_preprocessed.npz           # 执行运动和想象运动的预处理数据

4.4 加载预处理完成的数据​

import numpy as np

data = np.load('./out_data/EMM_task1_preprocessed.npz')
features = data['eeg']  # 特征数据，形状为(样本数, 通道数, 时间点数)
labels = data['labels']  # 特征数据标签，形状为（样本数，标签）,标签代表左手运动或右手运动

print(features.shape, labels.shape)
