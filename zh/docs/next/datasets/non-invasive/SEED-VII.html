5_SEED-VII

数据集简介与处理流程

数据集链接

https://bcmi.sjtu.edu.cn/~seed/seed-vii.html

1. 实验设计与数据采集​

受试者​

  • 人数：20 名参与者（10 男，10 女），年龄 19--26 岁（平均 22.5岁，标准差 1.8）。
  • 条件：全部为右利手，具有正常或矫正至正常的视力及正常听力。
  • 筛选：通过艾森克人格问卷（EPQ）筛选。

实验会话与试次​

  • 每位受试者完成 4 个会话（不同日期）。
  • 每个会话包含 20 个试次：
    • 第一部分：观看视频诱发情绪（视频时长 2--5 分钟）。
    • 第二部分：自我情绪评估（0--1 分值）。

情绪诱发设计​

  • 共 80 段视频：
    • 快乐、悲伤、恐惧、愤怒、厌恶、惊讶、中性（其中中性只有 8 段）。
    • 每个会话仅包含 7 种情绪中的 5 种，以避免情绪频繁切换。
  • 视频顺序经过精心安排，避免情绪突变。

2. 数据预处理流程​

EEG 信号​

  1. 坏道检查与插值 使用 MNE-Python 工具可视化检查 EEG 信号，对坏道进行插值修复。
  2. 滤波
    • 带通滤波：0.1--70 Hz
    • 陷波滤波：50 Hz（抑制电源工频干扰）
  3. 降采样 从原始采样率 1000 Hz 降至 200 Hz。
  4. 分段与特征提取
    • 时间窗：4 秒片段（不重叠）
    • 频带划分：
      1. δ (1--4 Hz)
      2. θ (4--8 Hz)
      3. α (8--14 Hz)
      4. β (14--31 Hz)
      5. γ (31--50 Hz)
    • 特征：差分熵（DE）
  5. 特征平滑 使用 LDS（线性动态系统）或滑动平均平滑特征。

3. 标签与同步​

  • 标签来源：每个视频的目标情绪类别（7 类）+ 主观评分（0--1）。
  • 时间对齐方式：EEG/眼动特征按照视频播放的时间窗对齐，每 4秒为一个样本。

4. 数据加载示例​

def load_data(session_id, sub_id):
  """
  Load a .npz data file for a specific session and subject ID.

  :param session_id: int, optional, default=1
      The session number, corresponding to the folder named "session{session_id}".
  :param sub_id: int, optional, default=1
      The subject ID used to filter files starting with "{sub_id}_".
  :return: numpy.lib.npyio.NpzFile or None
      The loaded npz file object if found; otherwise None.
  """
  for file in os.listdir(f"session{session_id}"):
    if file.startswith(f'subject{sub_id}') and file.endswith('.npz'):
      print(f"Loading data from: {file}(session={session_id})")
      return np.load(os.path.join(f"session{session_id}", file), allow_pickle=True)

5. 特征数据说明​

  • features：
    • 三维数组 [样本数, 频带数, 通道数]
    • 频带数固定为 5（δ、θ、α、β、γ）
    • 通道数固定为 62
  • labels：
    • 一维数组 [样本数]
    • 每个元素是情绪类别标签（0--6 或 0--3，取决于任务定义）
