13_Emotion

数据预处理流程说明文档

数据集链接

https://neurotycho.org/emotional-movie-task

1. 数据来源与目标​

本流程用于处理 128 通道 ECoG LFP 数据，并将其转换为可供后续分类/模式识别使用的 binned spike 表示。 数据包含：

  • 128 通道 LFP .mat 文件（每个通道一个文件）
  • 任务信息文件 Task_info.mat（包含试次标签、刺激序列）
  • 刺激开始时间文件 Movie_start_time.mat

处理目标：

  • 对 LFP 进行带通滤波（20–250 Hz）
  • 按试次切分，统一采样数
  • 降采样至指定时间步（target_timesteps）
  • 根据自适应阈值提取 spike
  • 进行时间 binning，得到 (trials, binned_time, channels) 格式的稀疏 spike 表
  • 保存为 .npz 格式，包含 spike 和 label 两个字段

2. 参数配置​

参数作用代码变量默认值

滤波高截止频率带通滤波上限 (Hz)highcut250

3. 处理流程​

3.1 标签与试次时间对齐​

  • 从 Task_info.mat 中读取 MovieSequence（刺激序列）
  • 建立 刺激 ID → 情绪类别 映射： {1: 恐惧, 2: 恐惧, 3: 威胁, 4: 威胁, 5: 中性, 6: 中性} 对应数值标签：0=Frighten, 1=Threaten, 2=Neutral
  • 从 Movie_start_time.mat 中读取每个 trial 的起始时间（ms）
  • 取两者最小长度，保证 trial 数对齐

3.2 LFP 裁剪与滤波​

  • 对每个通道：
    • 读取 LFP_ch{n}.mat
    • 按 trial 起始时间裁剪 trial_duration_sec 长度的片段
    • 使用 3 阶巴特沃斯带通滤波器（20–250 Hz）去除低频漂移与高频噪声
    • 使用 np.interp 线性插值降采样至 target_timesteps
  • 数据形状变化： 原始: (samples,) 结果: (channels, trials, target_timesteps)

3.3 Spike 检测（自适应阈值）​

  • 对每个 trial、每个通道：
    • 计算 LFP 均值 μ 和标准差 σ
    • 设定检测阈值：
      • 正阈值 = μ + 2σ
      • 负阈值 = μ - 2σ
    • 遍历时间步： 若信号超过正阈值或低于负阈值，则标记 spike 检测后进入不应期（refractory_ms），期间忽略检测
  • 结果： spike_data 形状: (trials, target_timesteps, channels) 值为 0/1

3.4 时间 Bin 操作​

  • 将 spike 序列按 bin_size 聚合
  • 每个 bin 内统计 spike 数量
  • 输出形状：(trials, binned_time, channels)
  • 例如： 原始: target_timesteps=50, bin_size=10 → binned_time = 50 // 10 = 5 结果 shape: (n_trials, 5, 128)

3.5 可视化​

  • Filtered LFP 曲线
  • 标签分布柱状图
  • Spike Raster Plot（点阵图）
  • Binned Spike 热力图

3.6 保存​

  • 保存至：processed/data_YYYYMMDD.npz
  • 包含：
    • spike：binned spike 数据，形状 (trials, binned_time, channels)，类型 uint8
    • label：情绪标签，形状 (trials, 1)，类型 int

4. 输出示例​

Trial 数: 60, 标签 shape: (60, 1)
LFP 处理完成，shape: (60, 50, 128)
Spike 生成完成，shape: (60, 50, 128)
NPZ 文件路径: .../processed/data_20100807.npz
包含字段：['spike', 'label']

5. 注意事项​

  • 滤波器频率设置：低截止 > 0 Hz，高截止 < Nyquist 频率
  • trial 截断：如果某 trial 末尾不足 trial_duration_sec 秒，会被丢弃
  • bin_size 需整除 target_timesteps，否则会丢弃最后不完整的 bin
  • 阈值检测灵敏度：可调整系数 2σ 为 1.5σ 或 3σ
  • 不应期单位：这里的 refractory_ms 在降采样后依然按“步”计算，需注意换算
