7_NPR M1

一、数据集概述​

本数据集来源于 Zenodo Record 3854034，包含非人灵长类（NHP）在执行中心外（center-out）光标控制任务时的初级运动皮层（M1）侵入式神经记录。神经信号通过 192 通道 Utah 阵列采集，行为数据包括二维光标位置以及对应的目标位置。每个 .mat 文件代表一个采集会话。

原始数据集包括：

  • Spikes：每个通道最多 3 个单元的排序尖峰时间。
  • Cursor position：二维光标位置，采样率为 24.4 kHz。
  • Target position：当前目标位置的二维坐标。
  • Timestamps：试次对齐的时间信息。

二、数据采集​

  • 被试：植入 192 通道 Utah 阵列的非人灵长类。
  • 任务：光标中心外指向任务（中心到目标点）。
  • 神经信号：每个电极最多 3 个排序单元的尖峰时间。
  • 行为信号：光标位置 (cursor_pos) 和目标位置 (target_pos)。
  • 采样率：行为信号 250 Hz。

三、预处理流程​

使用 preprocess.py 对原始 .mat 数据进行处理，得到适合回归解码模型的分箱（binning）与窗口化（windowing）数据。

1. 尖峰时间转换​

  • 提取所有通道、所有单元的尖峰时间。
  • 将同一通道的单元尖峰时间合并。
  • 按行为采样频率转换为 尖峰计数矩阵。

2. 试次切分​

  • 计算目标位置逐帧的位移量。
  • 当位移大于阈值（>1 单位）时，判定为新试次开始。
  • 按切分点将神经和行为数据划分为独立试次。

3. 时间分箱（Binning）​

  • 尖峰分箱：按固定长度 bin（bin_size，默认 5 个采样点）统计尖峰数。
  • 行为分箱：在相同 bin 窗口内计算光标位置均值，实现对齐。

4. 窗口化（Windowing for Regression）​

  • 滑动窗口（window_size，默认 100 bin）+ 步长（stride，默认 10 bin）。

  • 为每个试次生成成对的 (features, labels) 窗口：

    • Features：[window_size, n_channels] 尖峰计数。
    • Labels：[window_size, behavior_dim] 分箱后的光标位置。

5. 保存结果​

每个会话生成：

  • spikes.npy：形状 (n_samples, window_size, n_channels)
  • behaviors.npy：形状 (n_samples, window_size, behavior_dim)
  • 前 3 个试次的光栅图（.png）

四、预处理后文件结构​

dataset_root/
  session_1/
    spikes.npy
    behaviors.npy
    trial_0_raster.png
    trial_1_raster.png
    trial_2_raster.png
  session_2/
    ...

五、使用方法​

运行预处理脚本：

python preprocess.py \
    --data_root ./3854034 \
    --save_root ./out \
    --bin_size 5 \
    --window_size 100 \
    --stride 10

加载处理后的数据：

import numpy as np

spikes = np.load('./out/session_1/spikes.npy')
behaviors = np.load('./out/session_1/behaviors.npy')
