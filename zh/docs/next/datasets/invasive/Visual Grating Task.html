10_Visual Grating Task

数据预处理

预处理流程文档​

数据集​

访问数据集： 🔗 Visual Grating Task | neurotycho.org

1. 项目概述​

本项目提供了一个完整的 视觉光栅任务 (Visual Grating Task) ECoG 数据预处理流程，数据来源于 neurotycho.org。

目标是从原始 .mat 文件中提取事件相关片段，进行预处理（如 CAR 公共平均参考去噪、降采样），并保存为适合神经解码实验使用的结构化数据集。

参考文献： Neural Decoding Using Kernel-Based Functional Representation of ECoG Recordings

2. 主要特性​

  • 支持批量加载 visual_grating_task/data/ 中的多个 .mat ECoG 数据文件
  • 自动检测并提取两类事件：blank 与 raster
  • 执行 CAR 去噪，去除全局噪声成分
  • 将高采样率的 ECoG 信号段降采样至固定长度
  • 输出 .npz 文件，包含处理后的样本及元信息
  • 生成带统计信息的处理报告

3. 核心函数​

get_event_indices(mat_file_path)​

  • 功能：从事件标签通道中提取各类事件（blank, raster）的起止索引。
  • 输出：包含事件类型、分段索引及对应标签值的字典。

process_mat_file(mat_file_path)​

  • 功能：处理单个 .mat 文件，提取有效事件段，执行 CAR 去噪与降采样。
  • 输出：结构化字典，包括：
    • samples: (num_samples, 300, 128)
    • labels: (num_samples,)
    • label_names: ['blank', 'raster']
    • file_info: (num_samples,) 来源文件名
    • sample_label_names: 每个样本对应的事件名称

4. 使用方法​

1. 准备数据​

将 Visual Grating Task 数据集中的所有 .mat 文件放入：

visual_grating_task/data/

2. 运行预处理脚本​

python ecog_process_all2.py

3. 查看结果​

输出目录示例：

visual_grating_task/data/
├── visual_grating_task.npz        # 处理后的 ECoG 样本及元信息
├── processing_report1.txt         # 处理统计报告

5. 输出数据格式​

visual_grating_task.npz​

  • 类型：Numpy .npz 容器
  • 字段说明：
    • samples: [N, 300, 128] → N 个样本，每个 300 个时间点 × 128 个通道
    • labels: [N] → 整数类别标签（0=blank, 1=raster）
    • label_names: ['blank', 'raster']
    • file_info: [N] → 每个样本对应的源文件名
    • sample_label_names: [N] → 每个样本的可读标签名

6. 预处理步骤​

  1. 加载数据

    • 使用 scipy.io.loadmat 读取 .mat 文件
    • 访问矩阵 X：
      • X[:128, :] → 128 通道原始 ECoG 信号
      • X[128, :] → 每个时间点的事件标签

  2. 事件检测

    • blank：标签值在 [300, 400]
    • raster：标签值 >100 且不在 [300, 400]
    • 对每类事件提取连续片段索引

  3. CAR 去噪

    car_signal = signal_data - np.mean(signal_data, axis=0, keepdims=True)

  4. 片段选择

    • 仅保留长度 ≥1920 的片段
    • 截取 [start+20 : start+20+1900]，避免状态切换边缘影响

  5. 降采样

    • 将每段 1900 个采样点插值降采样至 300 点

  6. 样本组装

    • 每个事件格式化为 (300, 128)
    • 保存标签、类别名称和源文件信息

7. 事件标签定义​

标签值事件名称条件
0blank标签值 ∈ [300, 400]
1raster标签值 >100 且不在 [300, 400]

8. 处理统计信息​

生成的 processing_report1.txt 文件包含：

  • 总处理文件数
  • 有效样本总数
  • 每类标签的样本数
  • 每个原始文件贡献的样本数量

9. 常见问题排查​

  • MAT 文件读取失败：检查文件完整性及 MATLAB 版本兼容性
  • 未检测到有效事件：确认事件标签通道存在并符合预期编码
  • 输出形状异常：检查 .mat 文件是否均包含 X 矩阵且格式正确
