SearchAgent 搜索流程伪代码​

Algorithm: 验证驱动的架构搜索与集成选择

Input:
    训练数据 D_train = (X_train, y_train)
    验证数据 D_val = (X_val, y_val)
    任务类型 task_type in {classification, regression}
    搜索配置:
        max_models_per_round, max_search_rounds, max_depth,
        val_threshold, retain_top_k, ensemble_top_m

Output:
    最佳组合模型 f_best
    包含保留候选和所选集成的搜索摘要

1.  从本地模型库中发现候选模块。
2.  初始化保留候选集 P = empty。

3.  For round r = 1 to max_search_rounds:
4.      在搜索约束下采样最多 max_models_per_round 个候选架构。

5.      For each 候选架构 a:
6.          构建时序模型:
                InputProjection -> 搜索到的模块块 -> 任务头。
7.          在 D_train 上训练模型。
8.          在 D_val 上评估模型。

9.          If a 的验证指标 >= val_threshold:
10.             将 a 添加到 P。

11.     仅保留 P 中验证指标最高的 retain_top_k 个候选。
12.     If |P| >= retain_top_k:
13.         停止搜索。

14. If P 为空:
15.     使用最佳成功训练候选作为回退池。

16. 从 P 中选择 top ensemble_top_m 个候选。
17. 在 D_val 上评估候选集成:
        classification: 通过均值或 stacking 组合类别概率;
        regression: 平均预测轨迹。
18. 选择验证指标最佳的集成。
19. 将所选集成返回为 f_best。

外部评估:
20. 将 f_best 应用于每个留出的测试日。
21. 报告逐日指标、平均指标、中位指标和最低指标。

使用示例​


import bcijelly as bj
from bcijelly import SearchAgent

loaded = bj.load_data(
    dataset_name="5_Jango_force",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)

agent = SearchAgent(task_type="classification", device="cuda")

agent.fit(loaded.train_dataset, loaded.val_dataset)

by_day = {f"day_{i}": ds for i, ds in enumerate(loaded.test_day_datasets or [])}
results = agent.test(by_day)
print(results)
