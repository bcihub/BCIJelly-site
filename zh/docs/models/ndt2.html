10_NDT2

一、模型简介​

NDT2（Neural Data Transformer 2），也称为STPatch（Space-Time Patching Transformer），是NDT1的进阶版本，由International Brain Laboratory（IBL）团队提出。其核心创新在于：将神经脉冲数据同时在时间维度和神经元空间维度上进行分块，构建二维token网格，使Transformer能够在时空两个轴向上同时建模注意力关系。这一设计突破了NDT1仅在时间维度展开序列的局限，使模型能够显式地捕获不同脑区神经元之间的空间交互模式。

从代码实现与网络拓扑来看，本 NDT2 架构由以下核心模块构成：

1.时空分块器（Patcher）

原始脉冲数据（batch × time × neurons）首先被分块器在时间和神经元维度上进行二维分块：

  • 时间分块：将时间序列按 max_time_F 的窗口大小进行不重叠切分，生成若干时间patch。
  • 空间分块：将神经元轴按 max_space_F 的窗口大小进行不重叠切分，生成若干空间patch。
  • 最终每个patch包含一个时间窗口内一组神经元的脉冲活动，形成二维token网格（n_time_patches × n_space_patches）。

2.二维位置与区域嵌入

每个token被赋予三种位置信息：

  • 空间位置嵌入：可学习的空间patch索引嵌入，标识token在神经元轴上的位置。
  • 时间位置嵌入：可学习的时间patch索引嵌入，标识token在时间轴上的位置。
  • 脑区嵌入：可选的脑区标识嵌入，通过查找表将每个神经元映射到其所属脑区，使模型能够感知解剖学先验。

3.双重注意力掩码

NDT2的注意力机制同时受时间和空间两个维度的掩码约束：

  • 时间上下文掩码：控制每个token在时间轴上能关注的范围，支持因果、反因果与全双向模式。
  • 空间上下文掩码：控制每个token在神经元空间轴上能关注的范围。
  • 两个掩码通过逻辑与操作组合，形成二维注意力拓扑。模型还支持随机token掩码用于自监督预训练。

4.脑区查找表（RegionLookup）

模型内置了神经元到脑区的映射查找表。通过将每个神经元的实验会话与通道索引解析为脑区标签，模型能够在嵌入层和掩码策略中利用解剖学先验知识，例如实现跨脑区或脑区内的定向掩码。

5.线性解码头与多任务支持

与NDT1类似，NDT2根据任务类型配置不同的解码头：

  • 自监督重建：线性投影至patch内的原始维度，配合泊松或MSE损失。
  • 监督分类/回归：将所有patch的隐藏状态展平后馈入线性层，输出分类或回归结果。

三、调用示例​

classification​

import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.ndt2(task="classification")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

regression​

import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="regression",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.ndt2(task="regression")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

By default, pretrained weights are not loaded. To load pretrained weights, use:

train_result = model.train(
    loaded_data.train_dataset, loaded_data.val_dataset,
    pretrained_path="auto",
)

四、模型调试与评估​

评估指标

  • 分类任务：计算整体准确率，并生成混淆矩阵以诊断类别不平衡或错判偏好。
  • 回归任务：在自动逆归一化后的原始物理空间内，计算均方误差（MSE）、平均绝对误差（MAE）及决定系数。

核心调试技巧

  • Patch尺寸选择：max_time_F 与 max_space_F 共同决定了token的粒度。较小的patch提供更高的时空分辨率但增加序列长度，较大的patch则降低计算成本但可能损失局部细节。
  • 双向注意力拓扑：全双向注意力（context_forward=-1, context_backward=-1）使每个token能关注所有其他token，适合离线全局分析；设置有限窗口则模拟局部注意力，适合在线解码。
  • 脑区嵌入：当数据包含多脑区记录时，启用 embed_region 可使模型学习不同脑区的特异性表征，对跨脑区融合解码尤为关键。
  • 随机Token掩码：相比NDT1的神经元级掩码，NDT2的token级随机掩码在patch粒度上进行遮蔽，更适合捕获时空局部结构的重建任务。
  • Stitching机制：与NDT1类似，支持通过线性投影将不同神经元数量的输入映射至统一维度，实现跨会话模型复用。

五、应用说明​

NDT2 架构在以下场景中展现出独特优势：

(1) 全脑尺度神经解码：通过显式的空间分块与注意力机制，NDT2能够在数百甚至数千通道的全脑记录中自动发现脑区间的功能连接模式，对大规模神经群体编码的解码能力显著优于纯时序模型。

(2) 脑区特异性分析：脑区嵌入与空间掩码的结合，使研究者能够定向地分析特定脑区内部或脑区之间的信息流动模式，为探究运动皮层、前额叶等区域在运动规划与执行中的协同机制提供了强有力的工具。
