4_LSTM

一、模型简介​

LSTM（长短期记忆网络）是一种特殊的循环神经网络（RNN）架构，最早由Hochreiter和Schmidhuber于1997年提出。其核心目的是解决传统RNN在处理长序列数据时，梯度消失和梯度爆炸的问题。LSTM通过引入三个门控机制（输入门、遗忘门和输出门），允许网络在学习过程中保持或忘记信息，从而在长时间依赖的任务中发挥优势。

从代码实现与网络拓扑来看，本 LSTM 架构由以下核心模块构成：

1.时序序列编码器

原始输入通常为高维多通道神经或生理信号。模型底层依托于高度封装的RNNEncoder组件进行序列摄入。通过设定网络层数与双向标志位，模型能够构建深层的双向感受野，在当前时间步同时捕获历史上下文与未来演变趋势。

2.核心门控与状态流转机制

与 Transformer 依赖显式位置编码不同，LSTM 凭借其递归拓扑天然契合强时序依赖任务。通过遗忘门、输入门与输出门的精密协作，网络在漫长的时间步推演中，能够自主决定神经电位特征的“保留”或“遗忘”。这一核心门控机制有效攻克了传统 RNN 的梯度弥散瓶颈。

3.双轨任务解码头

为完美适配 BCI 等多场景范式，模型在输出端设计了高度灵活的分支解码逻辑

  • 全局池化分类解码：

针对离散的动作意图识别或整段信号定性分析，模型提供了灵活的池化策略。无论是对时间维度进行全局均值池化，还是精准提取序列末步状态，特征随后都会流入包含 LayerNorm 与 Dropout 的级联全连接层，输出最终的分类 Logits。

  • 时间分布式回归解码：

当面临连续运动学坐标解析等 Seq2Seq 回归需求时，模型采用专用的 TimeDistributedLinear 模块。该模块绕过时间池化，直接在长达序列长度的每一个时间步上，独立且共享权重地应用带 LayerNorm 的线性映射，实现对高频动态系统的逐帧高保真追踪。

三、调用示例​

classification​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.lstm(task="classification")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

四、模型调试与评估​

评估指标

  • 分类任务：计算整体准确率，并生成混淆矩阵以诊断类别不平衡或错判偏好。
  • 回归任务：在自动逆归一化后的原始物理空间内，计算均方误差、平均绝对误差 及决定系数 ( R2 )。

核心调试技巧

  • 强制梯度裁剪：在优化器步进前强制裁剪梯度，规避长序列训练中的梯度爆炸问题，提升模型鲁棒性。
  • 动态早停与最优权重快照：监控验证集表现以保存最优权重；在容忍周期内若无提升则提前终止，防止过拟合。
  • 动态学习率：采用学习率衰减策略或自适应优化器，平衡训练收敛速度与稳定性。

五、应用说明​

该 LSTM 实现框架在设计上表现出了极强的扩展性，能够灵活应对不同领域的核心痛点：

(1) 多维复杂神经解码应用：凭借对微弱动态特征的敏锐捕获与长程时序依赖挖掘能力，该框架能实现极高精度的时序模式识别，完美胜任运动意图解码（脑电控制源）、情感状态追踪以及默读语音识别。

(2) 灵活的在线/离线部署架构：框架具备高度的结构延展性：配置为单向模式时采用因果步进，实现极低延迟，完美适配实时闭环控制系统；而在双向模式下，则利用全局上下文获取极致的离线高精度解码表现。
