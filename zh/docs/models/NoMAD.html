15_NoMAD

一、模型简介​

NoMAD（Neural Dynamics Alignment Decoder）是一种基于LFADS（Latent Factor Analysis via Dynamical Systems）的动力学对齐解码框架。其核心思想在于：在LFADS学习到的低维神经动力学流形之上，叠加一个身份初始化的前馈对齐网络（Identity FFN Aligner），使模型能够在无需重新训练主干网络的情况下，通过仅微调少量对齐参数即可适配新的实验会话或被试。这一设计显著降低了跨天BCI系统的校准成本。

从代码实现与网络拓扑来看，本 NoMAD 架构由以下核心模块构成：

  1. NoMAD读入链（NoMADReadin）

原始高维神经信号首先经过读入链处理：

  • 低维线性投影：将原始输入维度投影至编码器维度（enc_dim），权重以正态分布初始化
  • LayerNorm归一化：对投影后的特征进行层归一化，稳定后续LFADS编码器的输入分布。
  • 身份FFN对齐器（IdentityFFNAligner）：一个三层全馈网络，所有权重初始化为恒等映射，偏置初始化为零，末尾附带LayerNorm。在跨天迁移时，仅需微调此对齐器即可将新会话的神经分布映射至LFADS主干的训练分布。
  1. LFADS动力学主干

对齐后的特征被馈入标准的LFADS变分序列自编码器：

  • IC编码器：反向RNN推断每次试验的初始条件分布。
  • 生成器：根据初始条件与学习到的非线性动力学规则自主演化。
  • CI编码器与控制器：推断外部扰动并注入生成器。
  • 泊松重构：将潜因子映射为神经元发放率。

所有LFADS组件的参数在跨天迁移时保持冻结，仅对齐器可训练。

  1. 行为解码头

LFADS的生成器状态被馈入行为解码头：

  • 双向GRU：在生成器状态序列上建模双向时序依赖。
  • 多层感知器头：两层全连接网络（含ReLU、Dropout）将GRU输出映射为二维运动学坐标。
  • 回归解码：逐帧输出连续运动轨迹预测。
  1. 分类解码头（可选）

对于离散意图识别任务，NoMAD提供分类变体：

  • 在生成器状态上进行时序池化（均值、末步或最大值）。
  • 馈入两层全连接分类头输出类别Logits。

三、调用示例​

classification​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.nomad(task="classification")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

四、模型调试与评估​

评估指标

  • 回归任务：在自动逆归一化后的原始物理空间内，计算均方误差（MSE）、平均绝对误差（MAE）及决定系数
  • 分类任务：计算整体准确率，并生成混淆矩阵。

核心调试技巧

  • 对齐器训练：align_epochs 参数控制跨天迁移时对齐器的微调轮数。通常5-10轮即可收敛；过多轮次可能导致对齐器偏离恒等初始化过远，破坏LFADS主干的动力学先验。
  • 泊松重构权重：rec_weight 参数平衡行为预测损失与神经活动重构损失。增大该值可增强潜表征对神经数据的保真度，但可能稀释行为预测的梯度信号。
  • 平滑正则化：smooth_weight 参数惩罚生成器状态的逐帧跳变，鼓励平滑的动力学轨迹，防止解码输出出现高频抖动。
  • 生成器/编码器维度：gen_dim 与 enc_dim 共同决定了LFADS的潜空间容量。较大的维度适合高维神经记录，但增加计算与过拟合风险。

五、应用说明​

NoMAD 架构在以下场景中展现出核心价值：

(1) 零校准跨天解码：传统BCI系统在跨天使用时面临严重的神经信号漂移问题，需要频繁重新校准。NoMAD通过冻结LFADS动力学主干并仅训练轻量对齐器的方式，实现了跨天的零校准或极低校准解码，大幅提升了BCI系统的临床可用性。

(2) 神经动力学迁移分析：身份初始化的对齐器提供了从新会话到参考会话的可解释映射关系。通过分析对齐器学习到的权重变化，研究者能够量化神经群体编码在不同天之间的漂移模式，为理解长期神经可塑性提供新视角。
