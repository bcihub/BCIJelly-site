9_NDT1

一、模型简介​

NDT1（Neural Data Transformer 1）是一种专为大规模神经群体活动建模设计的Transformer架构，最初由Ye和Pandarinath于2021年提出。与通用Transformer不同，NDT1深度针对神经数据的特殊结构进行了定制：它能够处理稀疏的脉冲序列、支持多种掩码预训练策略（自监督学习），并引入了旋转位置编码（RoPE）以更好地编码神经信号的时序关系。其核心假设在于：通过掩码-预测的自监督范式，Transformer能够从大规模未标注神经数据中学习到可迁移的神经动力学表征。

从代码实现与网络拓扑来看，本 NDT1 架构由以下核心模块构成：

  1. 神经嵌入层（NeuralEmbeddingLayer）

原始脉冲序列（batch × seq_len × n_channels）首先通过嵌入层进行表征：

  • 脉冲嵌入：支持线性投影与可学习嵌入表两种模式。嵌入表模式将离散脉冲计数映射为稠密向量，更适合处理稀疏脉冲数据。
  • 时间堆叠（Stacking）：可选地将连续若干时间步的嵌入拼接后投影，实现时间分辨率的降采样。
  • 位置编码：支持可学习位置嵌入与旋转位置编码（RoPE）两种方案。RoPE通过在Query和Key张量上应用旋转矩阵，将相对位置信息编码进注意力分数，更好地适应变长序列。
  • 提示标记与会话标记：可选地在序列首部拼接任务类型提示或实验会话标识，为模型提供任务上下文与跨天对齐信息。
  1. 旋转位置编码（RoPE）

NDT1实现了完整的旋转位置编码机制。通过预计算不同频率的正弦/余弦表，对Query和Key的每个头维度应用旋转变换。该编码方式使注意力分数自然地反映token间的相对距离，无需额外的位置嵌入参数，且天然支持变长序列。

  1. 上下文掩码与多策略掩码器

NDT1的注意力机制支持灵活的上下文掩码：

  • 前向/后向上下文窗口：通过 context_forward 与 context_backward 参数控制每个token能够关注的时间范围，支持因果（仅关注过去）、反因果（仅关注未来）与全双向三种模式。
  • 多策略掩码器：自监督预训练时，支持随机神经元掩码、因果掩码、脑区间掩码、脑区内掩码等多种掩码策略，迫使模型学习不同粒度的神经动力学表征。
  1. ScaleNorm与Fixup初始化

NDT1提供了ScaleNorm（基于可学习缩放因子的归一化）替代标准LayerNorm的选项，并实现了Fixup初始化策略——通过按层数缩放权重初始值，使深层Transformer在不使用Warmup的情况下也能稳定训练。

  1. 线性解码头与多任务支持

模型末端根据任务类型配置不同的解码头：

  • 自监督重建：线性投影至原始通道维度，配合泊松负对数似然或MSE损失。
  • 监督分类/回归：线性投影至输出类别数或目标维度。
  • CTC解码：适用于序列到序列的语音或文本解码任务。

三、调用示例​

classification​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.ndt1(task="classification")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

regression​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="regression",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.ndt1(task="regression")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

By default, pretrained weights are not loaded. To load pretrained weights, use:


train_result = model.train(
    loaded_data.train_dataset, loaded_data.val_dataset,
    pretrained_path="auto",
)

四、模型调试与评估​

评估指标

  • 分类任务：计算整体准确率，并生成混淆矩阵以诊断类别不平衡或错判偏好。
  • 回归任务：在自动逆归一化后的原始物理空间内，计算均方误差（MSE）、平均绝对误差（MAE）及决定系数。

核心调试技巧

  • 嵌入模式选择：对于脉冲计数数据，embed 模式通过可学习嵌入表将离散计数映射为连续向量，通常优于直接线性投影；对于连续值神经信号，linear 模式更为合适。
  • RoPE基频调节：rope_theta 参数控制旋转编码的基频，默认10000。增大该值可扩展模型可编码的序列长度。
  • 上下文窗口：在自监督预训练中，使用较窄的上下文窗口（如仅后向）可模拟因果预测任务；在下游微调中，全双向窗口通常性能更优。
  • 掩码策略：neuron 掩码随机遮蔽部分神经元的全部时间步，迫使模型学习跨神经元的群体动力学；causal 掩码则训练模型进行时序前向预测。
  • Stitching机制：当不同实验会话的神经元数量不一致时，NeuralStitcher 通过线性投影将不同维度的输入映射至统一的隐藏维度，实现跨会话模型复用。

五、应用说明​

NDT1 架架在神经科学与脑机接口领域具有深远的应用价值：

(1) 大规模神经基础模型：通过掩码-预测的自监督预训练范式，NDT1能够在大规模、多会话的神经数据集上学习通用的神经动力学表征。预训练后的模型可通过线性探测或轻量微调快速适配到下游任务，显著降低小样本场景下的数据需求。

(2) 跨天跨脑区迁移学习：Stitching机制与会话标记的结合，使NDT1能够在不同天、不同脑区甚至不同被试的神经数据间进行知识迁移，为长期稳定的BCI系统提供了技术路径。
