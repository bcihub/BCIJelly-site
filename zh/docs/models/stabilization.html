14_Stabilization

一、模型简介​

Stabilization是一种专为解决脑机接口（BCI）神经信号跨日或跨时间段非平稳性而设计的流水线模型。在植入式微电极阵列记录中，神经群体信号的分布会随时间推移发生漂移。该模型假设尽管观测层面的神经发放存在漂移，但其底层的低维神经动力学流形是相对稳定的。

从底层理论与工程实现来看，本模型的核心架构由以下五个关键部分构成：

  1. 高斯因子分析基准稳定器

    在训练阶段，模型提取训练数据末尾的特定数量试次，利用期望最大化算法拟合一个带有对角私有方差的标准高斯因子分析模型。该模块作为基准稳定器，负责从高维且包含噪声的神经通道中提取纯净的初始潜变量。

  2. 跨日动态流形校准

    在面对全新测试日的偏移数据时，模型拒绝从零重新训练。相反，它仅利用测试集最前端极少量的校准试次（受系统预设的比例或阈值控制），计算并更新当天的稳定器参数。这种少样本校准机制极大地降低了 BCI 系统的日常维护成本。

  3. 载荷矩阵正交变换与对齐

    为了消除不同天因子分析存在的旋转不确定性，模型通过比较基准载荷矩阵与更新后的载荷矩阵，依据预设的容差阈值与通道数自动识别稳定的载荷行。随后，系统求解最优正交变换矩阵（即普氏对齐算法），强制将测试日的潜变量无缝旋转映射回基准训练空间。

  4. 任务解耦的 MLP 解码网络

    在获取对齐且高度标准化的潜特征后，模型接入下游多层感知机（MLP）进行意图解码。针对不同范式，模型在底层进行了严格的代码解耦：

    • 离散意图分类：调用标准的分类网络进行非线性决策边界划分。
    • 连续运动学回归：为实现高保真轨迹追踪，回归任务专门重构了回归网络，采用比例共轭梯度法进行极其高效的权重寻优。

  5. 目标驱动的解码器微调

    特别是在连续回归任务中，为了进一步挖掘解码性能，模型原生支持权重继承与微调机制。在完成正交对齐后，模型将原始训练数据与当前的校准数据拼接，对解码器的权重进行指定轮数的二次优化，以实现对微小非线性偏移的最终补偿。

二、调用示例​

分类任务​

import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.stabilization(task="classification")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

回归任务​

四、模型调试与评估​

在非平稳系统的性能评估中，模型内置了以下自动化诊断与可视化输出：

多维度回归评价：除全局决定系数外，强制解耦输出各独立维度的拟合优度，以精准排查特定方向的解码失效。

优化轨迹监控： 自动绘制训练损失曲线，直观判断共轭梯度下降过程中是否出现震荡或提前收敛。

时序跨日鲁棒性检验：按日期自然排序，输出含均值基准线的跨日指标趋势图，直观展现流形对齐算法的抗衰退能力。

五、应用说明​

Stabilization的理论机制直击长期植入式 BCI 的核心痛点，展现出极高的科研与临床应用价值：

(1) 对抗信号漂移与极低成本校准：通过本征流形提取与正交旋转对齐，有效补偿长期植入带来的电极微动与信号漂移。系统每日仅需极少量数据进行基线预热和矩阵更新，即可直接恢复高水平解码状态，彻底免除耗时的全模型重训。

(2) 赋能极低延迟的在线实时解码：架构实现了离线校准与在线解码的优雅切分。稳定矩阵在极短的预备阶段锁定后，下游解耦网络即可释放极致的推理速度，赋予系统强大的实时闭环控制能力与极低延迟的假肢操控体验。
