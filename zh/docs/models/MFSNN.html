16_MFSNN

一、模型简介​

MFSNN（Multi-Frequency Spiking Neural Network）是一种专为神经信号分类任务设计的脉冲神经网络（Spiking Neural Network, SNN）架构。与传统人工神经网络不同，MFSNN借鉴了生物神经元的脉冲发放机制，通过可微分的LIF（Leaky Integrate-and-Fire）发放率近似，在保持SNN生物合理性的同时实现了端到端的梯度训练。其核心假设在于：多频率时序通道分组与通道注意力机制的结合，能够有效捕获神经信号在不同时间尺度上的特征模式。

从代码实现与网络拓扑来看，本 MFSNN 架构由以下核心模块构成：

  1. LIF脉冲发放率近似层

模型底层依赖于可微分的LIF神经元模型。当系统安装了 SpikingJelly 库时，模型调用其原生LIFNode实现ATan代理函数的脉冲模拟；在缺少依赖的环境中，则自动降级为Sigmoid软激活近似，确保代码的可移植性。该层通过多步仿真（默认20步）对输入进行时间维度的发放率编码，将连续值转化为脉冲频率表征。

  1. 多通道分组子网络（SubNet）

输入通道被均匀划分为若干组（默认16组），每组独立构建一个子网络（SubNet）。每个SubNet内部包含：

  • 空间注意力分支：通过GroupNorm → LIF激活 → 通道注意力（Channel Attention）模块，自适应地为各通道分配权重。通道注意力采用全局平均池化后的两层全连接结构，中间嵌入LIF脉冲层以保持生物脉冲特性。
  • 时序卷积分支：采用膨胀因果卷积（Dilated Causal Convolution, dilation=2, kernel_size=3）提取多尺度时序特征，随后进行平均池化降采样。
  • 可学习时间映射矩阵：通过一个可训练参数矩阵将原始时间维度直接映射到降采样后的时间维度，与时空融合输出相加，提供额外的残差时间建模能力。
  1. 时空融合与分类解码头

各SubNet的输出在特征维度上进行拼接，经过LIF激活、LayerNorm归一化与Dropout正则化后，馈入全连接分类层输出最终的分类Logits。该设计使模型能够并行捕获不同频率带的神经活动模式，并在决策层进行跨组信息融合。

三、调用示例​

classification​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.mfsnn(task="classification")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

四、模型调试与评估​

评估指标

  • 分类任务：计算整体准确率，并生成混淆矩阵以诊断类别不平衡或错判偏好。

核心调试技巧

  • LIF时间常数调节：tau 参数控制LIF神经元膜电位的泄漏时间常数，较大值使神经元对长时程输入更敏感，较小值则聚焦于瞬态响应。
  • 仿真步数：steps 参数决定LIF层的仿真迭代次数，更多步数提供更精确的发放率估计但增加计算开销。
  • 通道分组数：groups 参数需整除输入通道维度，分组数越多则每个子网络处理的通道越少，有利于学习通道间的局部相关性。
  • 脉冲编码与软激活切换：通过 use_spikingjelly 开关可在真实脉冲仿真与Sigmoid近似间切换，前者更接近生物真实但需要额外依赖。
  • 损失函数选择：支持MSE（one-hot目标）与交叉熵两种损失类型，MSE损失在脉冲网络中通常提供更平滑的梯度信号。

五、应用说明​

该 MFSNN 实现框架在设计上充分利用了脉冲神经网络的固有优势：

(1) 低功耗事件驱动计算：SNN的脉冲稀疏性使其在神经形态硬件（如Intel Loihi、IBM TrueNorth）上具有天然的低功耗优势。MFSNN的多频率分组架构进一步增强了脉冲的稀疏性，使其非常适合边缘端BCI设备的实时部署。

(2) 多尺度时序特征捕获：通过通道分组与膨胀卷积的协同设计，模型能够在不同时间分辨率上同时提取神经信号的快速瞬态响应与缓慢调制趋势，对运动意图解码、情感状态分类等多类BCI任务具有广泛适用性。
