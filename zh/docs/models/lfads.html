12_LFADS

一、模型简介​

LFADS 是一种专为神经科学数据设计的变分序列自编码器。在脑机接口（BCI）与计算神经科学领域，单次试验的神经脉冲数据往往伴随极高的泊松噪声。LFADS 突破了传统高斯平滑或试次平均方法的局限，其核心假设在于：复杂的神经群体活动是由一个低维的、非线性的内在动力学系统所驱动的。

从理论机制来看，LFADS 的核心架构由以下五个关键部分构成：

  1. 变分序列自编码架构

    LFADS 将循环神经网络（RNN）强大的时序建模能力与变分推断的概率框架进行深度融合。它不再输出单一的确定性轨迹，而是学习隐变量的后验概率分布，从而在极高噪声的单脉冲数据中实现了前所未有的鲁棒性与泛化能力。

  2. 初始条件推断

    在每次运动或认知任务的起点，大脑的动力学系统均处于特定的初始状态。LFADS 利用一个反向运行的 RNN 编码器，通过读取整段神经序列的历史信息，精准推断出驱动当前试次动力学演化的初始条件及其分布概率。

  3. 自主动力学生成器

    作为 LFADS 的核心状态转移模块，在接收到初始条件后，生成器（通常为多层 RNN）会根据自身学习到的非线性动力学规则在时间尺度上自主向前演化。这一过程有效刻画了运动皮层在执行动作时所表现出的内在低维流形特性。

  4. 外部扰动推断与控制器

    为应对大脑活动中不可预见的外部刺激或未被记录脑区的反馈输入，LFADS 创新性地引入了控制器输入编码器与 RNN 控制器。这二者在每个时间步动态推断外部扰动，并将其作为时变输入实时注入生成器，从而显著提升了模型对非自主性突发神经活动的拟合能力。

  5. 泊松发射与发放率重构

    LFADS 的最终输出并非直接拟合离散的脉冲序列，而是将生成器的高维状态映射为低维的潜因子，随后通过指数映射（以确保非负性）还原出底层真实且平滑的神经元发放率（Firing Rates）。模型采用泊松负对数似然作为重构目标，严格契合了神经脉冲发放的统计物理规律。

三、调用示例​

classification​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.lfads(task="classification")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

四、模型调试与评估​

在神经科学领域，LFADS 的性能评估无法单纯依赖传统的均方误差或交叉熵，而必须紧密结合其泊松发射特性：

单脉冲比特数（BPS）： 脉冲序列处理的黄金标准。基于泊松负对数似然，衡量重构发放率较基线模型的信息增益；值越高，表明捕获的神经动力学细节越丰富。

PSTH 重构保真度： 用于验证去噪有效性。将推断的单试次发放率均值与经验 PSTH 计算决定系数（ R2 ）；优秀模型能在保持单次变异性的前提下，高保真还原 PSTH。

后验方差监控： 模型调试的关键。需密切监控初始条件与控制器输出的后验方差；若方差坍缩至下限，表明模型未有效学习不确定性，需及时调整 KL 退火策略。

五、应用说明​

依托上述五大核心机制，LFADS 成功破局了现代 BCI 领域内数个棘手的理论与工程难题： 这里为您将四大核心机制浓缩整合为最具代表性的两点：

(1) 高保真单次解码与多范式部署：利用动力学生成器，LFADS 能从单次高噪脉冲中精准提取平滑发放率，大幅提升意图解码信噪比。其非因果特性使其在离线高精度分析中表现卓越；若应用于实时闭环，则可通过“块解码”策略在控制延迟与平滑保真度间取得最佳平衡。

(2) 长期系统稳定与未知扰动推断：通过提取并对齐跨天的低维动力学流形（潜因子），有效对抗神经信号的长期非平稳漂移，免除频繁校准成本；同时借助控制器机制，能够逆向推断未被记录的外部脑区干扰，为探究全脑连通性提供关键工具。
