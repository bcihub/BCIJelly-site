6_FENet

一、模型简介​

FENet 是一套专为时序信号设计的独立回归包装器（Standalone FENet regression wrapper）。其核心理念在于通过受小波变换启发的网络架构，解决多通道时间序列中的特征提取与降维问题。它能够高效地在保留全局上下文的同时，分离并提取时序数据中的关键频段/尺度特征。从代码实现与网络拓扑来看，本 FENet 架构由以下核心模块构成：

1.小波特征分裂块 (WaveletBlock) 作为底层基础组件，该模块执行受 FENet 启发的单通道特征/传递分裂（single-channel feature/pass split block）。输入数据被填充并分别送入平行的 1D 卷积层（feat_conv 与 pass_conv）。其中特征提取分支通过 LeakyReLU 激活与自适应平均池化进行压缩，而传递分支则应用 Dropout 操作并继续向后层网络传递，从而在深度网络中实现逐层的信息解耦与提取。

2.通道共享骨干网络 (FENetBackbone) 在处理形如 (B, T, C) 的多通道生理数据时，框架采用了一种特殊的隔离拓扑：它将完全相同的单通道卷积堆叠结构，独立且重复地应用于每一个独立的神经信号通道（apply same single-channel stack to every neural channel）。各通道在经过多级 WaveletBlock 过滤后，提取到的多尺度特征会在最终阶段被拼接合并，并通过 LayerNorm 进行全局特征的标准化。

3.多模式序列解码头 (Dual-Track / Sequence Decoder Head) 为适配多种时序任务，FENetRegressorModel 配备了极具弹性的解码路由，可通过 sequence_decoder 参数在三种模式间自由切换： 全局解码 (Global)：将骨干提取的特征展平后，送入由全连接层（Linear）、GELU 激活及 Dropout 组成的多层感知机（MLP）头部，直接输出降维后的全局预测结果。 时序/混合解码 (Temporal/Hybrid)：专门配备了时间维度解码头（temporal_head），通过级联的 1D 卷积（Conv1d）代替全连接层，以保持时序分辨率不变。在 Hybrid 模式下，全局静态特征预测还可以与时序动态预测直接相加，满足时间分布式输出的复杂需要。

三、调用示例​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="6_FENet",
    protocol="cross_day", task="regression",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.fenet(task="regression")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

四、模型调试与评估​

FENet 模型的调试与评估深度集成在其专用的训练与验证循环器（regressor.py）中：

评估指标：

回归任务：该框架高度专注于连续变量的回归追踪。在模型验证阶段，除了依赖 SmoothL1Loss 计算梯度，系统还会严格计算均方误差（MSE）、平均绝对误差（MAE）以及决定系数（R2 Score）。其中，验证集的 R2 分数（best_val_r2）被框架确立为评估模型当前状态是否达到“最优”的绝对核心指标。

调试技巧：

强制梯度裁剪 (Mandatory Gradient Clipping)：为保障深度时序网络在复杂数据下的训练稳定性，底层的 _train_one_epoch 函数在每次执行优化器步进前，内置了强制性的 clip_grad_norm_ 操作，将参数的最大梯度范数严格限制为 1.0。

动态早停与最优权重快照 (Dynamic Early Stopping & Best Checkpoint)：内置了训练容忍度（patience）监控机制。当验证集的 R2 表现持续特定 Epochs 没有突破历史最佳时，系统会果断触发早停终止训练。同时，系统在训练期间实时克隆并序列化最优的模型权重（best_state_dict），确保在训练结束前自动回滚到性能峰值状态（restore_best）。

余弦退火学习率 (Cosine Annealing LR)：结合稳健的 AdamW 优化器，学习率调度器采用了 CosineAnnealingLR，允许学习率在给定的 Epoch 周期内平滑衰减至 eta_min（设定为初始学习率的 10%），有效避免了模型在训练后期于最优点附近产生震荡。

运行时自适应校准 (Test-time Calibration)：除了标准的“训练-测试”分离，该框架还创新性地支持了测试集的在线校准功能。在测试阶段，通过设置 calib_epochs 和 calib_ratio 参数，模型可以利用极少量的校准试验数据实施微调更新。为了防止灾难性遗忘，调试时还可以使用 calib_scope 精细控制仅解冻网络特定部分（如仅更新解码器 head 或是全网络 all）的梯度。

五、应用说明​

结合源代码中内置的数据验证断言和输入张量处理逻辑，FENet 的架构设计主要锚定以下前沿应用场景：

  • 高保真二维运动学轨迹回归 (2D Kinematic Trajectory Decoding)： 数据加载逻辑中存在严苛的检查点：目标真实值必须具有 (N, T, 2) 的形状（如触发报警 Regression y must have shape (N, T, 2)），且输入数据的默认识别标签名为 "neural_activity" 或 "spike"。这表明该模型完美契合于脑机接口（BCI）领域，能够利用受试者的神经元放电或特征时间序列，逐帧且连续地预测二维平面上的运动学坐标（如屏幕光标控制、二维速度矢量映射等）。

  • 弱耦合多通道传感器特征提取 (Uncoupled Multi-channel Sensor Mining)： 考虑到 FENetBackbone 放弃了在通道间做密集的空间混合，而是严谨地对每一个输入通道套用“共享参数的单通道滤波器池”，这非常适合处理物理空间上高度不相关或独立布置的传感器阵列。例如从广域分布的微电极阵列或高密度肌电（HD-sEMG）中，无偏倚地隔离并提取时频特征。

  • 跨日/非平稳生理在线自适应控制 (Cross-day Non-stationary Adaptive Control)： 由于神经生理信号在不同自然日之间容易产生基线漂移和极强的非平稳性。该模型预置了诸如 day_mean_std 这样的针对性预处理归一化策略。结合前述的 Test-time 校准功能，它能够在模型部署后，仅使用当天几秒钟到几分钟的新试验数据，对网络特定头部进行快速对齐与微调，大幅提升了 BCI 系统的可用性与在线鲁棒性。
