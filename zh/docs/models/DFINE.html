3_DFINE

一、模型简介​

DFINE 是一套专为处理复杂时序动态系统（如神经群体信号）设计的独立回归与状态推断框架。其核心理念在于将深度学习的非线性特征提取能力与经典线性动态系统（如 Kalman Filter/Smoother）的严谨数学推演相结合，在对高维数据进行降维的同时，显式地捕捉潜在状态的时序演变规律，从而实现极高鲁棒性的连续轨迹追踪。

从代码实现与网络拓扑来看，本 DFINE 架构由以下核心模块构成：

  1. 多层感知机特征映射网络

模型底层并未依赖传统的卷积或循环网络，而是采用了多组高度灵活的多层感知机（MLP）组件。其中，encoder 负责将高维的原始观测序列非线性投影至低维潜在空间（a_hat）；decoder 负责从潜在状态重构原始观测以保证核心信息不丢失；而专用的 mapper 网络则将平滑后的内部隐状态映射为最终的外部行为或运动学变量（behv_hat）。

  1. 核心线性动态系统模块

作为捕捉时序依赖的“心脏”，该架构深度集成了一个可微分的线性动态模型（LDM）。该模块内建了严格的状态转移矩阵（A）、发射矩阵（C）以及可学习的对角协方差参数。它在计算前向传播时执行类似卡尔曼滤波（Kalman Filtering）的操作，并在开启 do_smoothing=True 时执行反向平滑。这种机制允许网络在充满噪声的观测数据中，推断出极度平滑且符合物理/生理动力学演化规律的隐状态轨迹。

  1. 多步前瞻与多目标联合解码机制

在优化目标上，模型绕过了单一的回归损失，采用了精心设计的复合优化策略。损失函数不仅包含观测重构误差（recon_mse）和行为解码误差（behv_mse），还创新性地引入了基于状态转移矩阵的 K 步前瞻预测误差（pred_mse，由 steps_ahead 控制）。这迫使模型在隐空间内学习到具有极强前瞻推断能力的内部动态表示。

二、调用示例​

回归任务​

import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="regression",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.dfine(task="regression")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

四、模型调试与评估​

DFINE 模型的调试与评估核心集成于 regressor.py 中：

核心评估指标（多维度回归诊断） 同步计算原始量纲下的均方误差、平均绝对误差及决定系数 (R2)。其中，验证集 R2 (best_val_r2) 是触发早停与最优权重快照的绝对核心基准。

核心调试技巧

  • 强制梯度裁剪：优化器步进前强制执行 clip_grad_norm_，规避梯度爆炸，极大提升低信噪比数据下的训练稳定性。

  • 阶跃学习率与动态早停：配合 Adam 优化器采用 StepLR 周期性衰减学习率以防震荡；同时内置基于容忍度的早停机制。

  • 运行时自适应校准：测试阶段支持在线微调。通过 calib_scope 精细控制解冻范围（如浅层 mapper），利用极少量数据 (calib_ratio) 进行快速对齐，有效防止灾难性遗忘。

五、应用说明​

结合 DFINE 的代码逻辑与输入张量断言（强制要求回归目标具有 (N, T, 2) 的形状），该架构主要锚定以下前沿应用场景：

(1) 高保真连续解码与潜在动力学发掘：借助 LDM 提取平滑隐状态并显式拟合状态转移矩阵，模型不仅能有效滤除噪声，输出高度平滑的二维连续运动控制轨迹，更能作为计算显微镜，发掘并可视化底层的低维神经动力学流形。

(2) 跨日非平稳在线极速自适应：针对最棘手的跨日信号基线漂移，内置归一化策略并支持 Test-time 在线极速校准。临床部署前仅需极少量数据微调表层网络，有效兼顾核心动力学的长期记忆与映射的即时灵活性，大幅提升系统生命周期。
