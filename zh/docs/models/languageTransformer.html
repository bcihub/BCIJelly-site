7_Language Transformer

一、模型简介​

Language Transformer 是一种专为高频神经电生理信号解析与连续语音解码（Neural Speech Decoding）构建的端到端深度架构。在脑机接口（BCI）领域，尤其是面对复杂序列到序列（seq2seq）的映射任务时，传统的递归神经网络（如 LSTM）往往受制于时间步的串行计算瓶颈与极长序列下的梯度衰减。

本架构彻底摒弃了马尔可夫假设下的时序递归范式，引入了纯粹的 全局多头自注意力机制。结合 11_Speech 的联结主义时间分类（CTC）推断框架，模型能够在处理单次语音试次（Single-trial）时，瞬间捕获整段神经序列中任意两个时间步之间的非线性长程依赖，实现从高维时空动力学流形到离散语义符号的并行解码。

从理论机制与网络拓扑来看，本架构由以下核心模块构成：

  1. 前端过滤与局部流形展平

    神经群体活动往往伴随高频的瞬时泊松噪声。模型底层的 SpeechFeatureFrontend 并非简单透传信号，而是执行了一套严密的物理学预处理：

    • 高频噪声过滤：利用 GaussianSmoothing 进行一维卷积平滑，在时域上滤除高频伪影。
    • 跨天域偏移补偿：内置 day_weights 与 day_bias 参数矩阵，通过仿射变换在输入端执行跨天特征空间的线性校准。
    • 局部感受野重构：应用 Softsign 激活后，利用 nn.Unfold 将一维时序信号切分为重叠的二维窗口特征。这一操作极大地弥补了 Transformer 捕捉局部特征能力较弱的短板。

  2. 高维投影与绝对时序注入

    由于 Transformer 天生缺乏处理序列顺序的归纳偏置，前端输出的特征会首先通过线性投影对齐至统一的隐藏流形维度（d_model）。随后，模型引入基于正弦与余弦函数的位置编码，将绝对的时间节律信息无缝注入到神经特征中，赋予了模型解析脉冲序列时间演化规律的能力。

  3. 深层全局注意力编码器

    模型的核心计算引擎由多层 TransformerEncoderLayer 堆叠而成。

    • 多头注意力：允许模型在不同的特征子空间中，同时关注发音准备、肌肉激活与声学反馈等多个生理维度的时序依赖。
    • 前置归一化架构：显式设置 norm_first=True。在极深的拓扑结构中，这种设计被证明能极大提升残差连接（Residual Connections）中梯度的反向传播效率，保障训练的绝对稳定。

  4. 异步序列解码与边缘化推断

    模型输出端映射至词表空间后，引入 nn.CTCLoss。该机制通过概率边缘化，允许在未对齐的输入脉冲序列与目标文本序列之间进行端到端训练，彻底打破了传统解码器对帧级别强制对齐的依赖。

三、调用示例​

Speech​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="15_Speech",
    protocol="train-test", task="speech",
    split_ratio="7:1:2",
)
model = bcijelly.model.transformer(task="speech")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

四、模型调试与评估​

在连续语言解码领域，模型评估聚焦于全局序列拓扑保真度：

  • 序列编辑距离评估：弃用交叉熵，以字符错误率（CER/Levenshtein距离）为核心指标，通过最小编辑次数直观反映系统的真实可用度。

  • 强制动力学稳定：每次步进严格执行 clip_grad_norm_（阈值 1.0）并辅以 Norm-First 架构，构建双重防线以规避自注意力网络的梯度异常。

  • 自适应退火与泛化防御：集成 ReduceLROnPlateau 调度器实现学习率减半（factor=0.5）退火；配合验证集监控与最优快照（best_model.pt）回滚机制，完美阻断过拟合导致的泛化坍塌。

五、应用说明​

Language Transformer 架构凭借其全局注意力机制，在时序神经解码领域展现出了降维打击般的应用潜力：

(1) 打破对齐桎梏的自然语言直接生成：借助 CTC 机制与全局视野，模型彻底摆脱对精确“帧-音素”级标签的依赖，支持以整句文本直接训练。这大幅降低了神经数据标注成本，使“大脑意图直译自然语言”的实时语音系统成为现实。

(2) 高保真特征溯源与跨日漂移鲁棒性：利用多头注意力机制精准过滤瞬时噪声并聚焦核心语义簇，实现高保真映射；同时凭借极强的全局抽象能力，强制提取真正的“神经语义不变量”，有效对抗非平稳的信号基线漂移，保障长期跨日解码的稳定。
