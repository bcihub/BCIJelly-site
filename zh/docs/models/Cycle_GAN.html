13_Cycle-GAN

一、模型简介​

Cycle-GAN（循环生成对抗网络）在此处被创新性地引入为一种无监督的神经域自适应对齐器。在脑机接口（BCI）的长期应用中，微电极阵列记录的神经脉冲数据常常因电极微小位移或神经元自然演变而产生特征空间的非平稳漂移。Cycle-GAN 突破了传统必须依赖目标域大量标签重新训练下游解码器的局限，其核心机制在于：通过构建双向映射，将目标日漂移后的神经脉冲特征无缝对齐回源日的特征空间，从而实现预训练解码器的免重训复用。

从理论与代码架构来看，该 Cycle-GAN 对齐系统由以下四个关键模块构成：

  1. 双向神经特征生成器

    模型实例化了两个对称的生成器，均采用三层多层感知机架构，并融合了失活与激活机制。生成器负责学习高维神经特征流形之间的非线性映射，将源域特征转换为目标域特征，或将目标域特征还原至源域。

  2. 对抗鉴别器

    系统包含两个独立的鉴别器模块（也是三层结构，输出单节点），分别负责区分真实神经特征与生成器伪造的对齐特征。通过计算对抗损失，鉴别器与生成器在极小极大博弈中驱动对齐特征逼近真实的神经放电分布。

  3. 循环一致性与身份保留约束

    为防止生成器在映射高维神经脉冲时丢失核心的运动学或意图信息，模型强制引入了两种关键惩罚机制：

    • 循环一致性损失： 确保特征经过双重映射（从源域到目标域，再回到源域）后，能精确重构原始输入。
    • 身份保留损失： 当生成器输入同域特征时，约束其输出保持不变，避免对基底神经信号产生过度扭曲。

  4. 解耦的下游解码器

    系统解除了对齐器与解码器的耦合限制。提供用于拟合连续运动学信号（如力度、肌电图）的回归器（采用平滑损失），以及用于离散动作/状态识别的分类器（采用交叉熵损失，并内嵌梯度裁剪机制以防爆炸）。

三、调用示例​

classification​

import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.cycle_gan(task="classification")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

四、模型调试与评估​

本系统构建了严密的阶段性验证与指标追踪体系，模型调试聚焦于以下泛化评价：

  • 回归任务多维评估： 核心指标为多输出方差加权决定系数。针对二维运动学解码，额外拆分并输出独立坐标轴的逼近度。

  • 分类任务准确率监控： 聚焦分类准确率，其提升直接反映特征流形的有效重合度。

  • 周期性验证与断点持久化： 特定迭代后切入验证模式，追踪验证集（经对齐输入冻结的源域解码器）的表现。自动保存表现最优的生成器与解码器权重。

  • 全链路日志与后处理： 逐日汇总验证结果至表格文件；支持通过脚本将无表头数据矩阵无损映射回张量文件，适配后续分析。

五、应用说明​

依托非配对域自适应能力，本 Cycle-GAN 架构精准解决了当前长期植入式 BCI 领域的最痛点问题：

(1) 无监督的零重训对齐校准：系统摒弃了修改庞大下游解码器的传统思路，转而利用无监督的 Cycle-GAN 架构将偏移输入拉回初始特征空间。这使得系统无需依赖带标签的校准数据，仅靠少量无标签脑电序列即可后台完成自适应对齐，彻底免除逐日昂贵的模型重训成本。

(2) 跨任务泛化与离在线解耦部署：统一的特征对齐器展现出极强的通用性，可无缝驱动高度异构的下游任务（如回归或分类）。架构在设计上实现了完美解耦：极其耗时的生成对抗训练被限制在离线阶段，一旦权重固化，实时映射的数据流即可零延迟送入下游网络，保障了高鲁棒的在线实时闭环运行。
