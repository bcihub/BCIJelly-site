19_UniBCI

一、模型简介​

UniBCI（Universal Brain-Computer Interface）是一种基于混合注意力机制的侵入式神经信号通用解码模型，其骨架网络脱胎于 POYO-1（Azabou et al., 2024）的大规模神经群体活动预训练架构。在脑机接口（BCI）领域，传统单任务解码器在跨脑区、跨物种、跨行为范式泛化方面存在明显瓶颈。UniBCI 的核心假设在于：通过将任意形态的脉冲序列统一转化为固定长度的 Token 表示，并结合组内（Intra-patch）与组间（Inter-patch）双阶段注意力机制，模型能够在统一的表征空间中同时捕获神经活动的局部时序动态与全局跨通道交互，从而在分类（离散意图识别）与回归（连续运动解码）两类任务上共享同一个预训练骨架。

从代码实现与网络拓扑来看，本 UniBCI 架构由以下核心模块构成：

  1. 脉冲分词器（Spike Tokenizer）

模型不对原始脉冲序列进行手工特征提取，而是通过一套统一的 Token 化流程将任意形态的神经数据转化为固定维度的序列张量：

  • 通道分组：将 C 个神经通道均匀分配至 S 个空间 Token（默认 S=8），每个 Token 容纳 N=group_size 个神经元（默认 32），总容量为 S × N = 256 个通道。不足则补零，超出则截断。
  • 时序分块：通过均匀时间分箱（默认以均值池化聚合），将原始 T 个时间步压缩或填充至 P×W 的固定长度（P=patch_num=10, W=patch_size=10, 合计 100 个时间步）。
  • 位置编码生成：自动生成时间位置索引（time_pos）与空间位置索引（space_pos），同时为不足容量通道生成 padding_mask，供后续注意力模块进行选择性掩码。
  1. 输入投影与时空位置编码

分组后的脉冲张量 (B, P, W, S, C) 通过两层线性投影（C → embed_dim → embed_dim，含 ReLU 激活）将二进制脉冲映射为稠密实数表征。随后，模型注入两个独立的可学习位置嵌入表：

  • 时间嵌入（Time Embedding）：nn.Embedding(max_t, embed_dim)，为每个时间步提供绝对时序信息。
  • 空间嵌入（Space Embedding）：nn.Embedding(max_s, embed_dim)，为每个空间 Token 提供通道组身份信息。
  • 两种嵌入与输入特征直接相加后经 Dropout 正则化，使模型能够感知脉冲发生的"何时"与"何处"。
  1. 可选文本条件编码

模型支持通过文本元信息（物种、个体、脑区、任务类型、实验会话）对表征进行条件调制。具体而言，使用冻结的 SentenceTransformer（MiniLM）将自然语言描述编码为 384 维文本向量，再经冻结的两层投影网络（Linear → LayerNorm → ReLU）映射至 embed_dim，以加法方式注入 Token 序列。此设计使模型能够在多数据集联合预训练中感知数据来源的上下文差异。

  1. Intra-Inter 混合注意力编码器

核心骨干由多个 IntraInterBlock（默认 4 层）堆叠而成，每个块依次执行两个互补的注意力操作：

  • Intra-patch 线性注意力：在每个 Patch 内部沿时间维度 W 执行线性注意力（Linear Attention），采用 elu(x)+1 特征映射核，将标准点积注意力的 O(n²) 复杂度降至 O(n)，使模型能够高效处理 Patch 内的细粒度时序依赖。
  • Inter-patch 滑动窗口注意力：在 Patch 维度 P 上执行带因果掩码的滑动窗口注意力（Sliding Window Attention），窗口大小默认 10。使用标准的缩放点积 Softmax 注意力，使每个 Patch 仅关注其邻近的前序 Patch，兼顾全局感受野与计算效率。
  • 特征融合：Inter-patch 特征被广播扩展至时间维度，与 Intra-patch 输出相加后经残差连接与 FFN（两层全连接，扩展比 4.0，ReLU 激活），实现跨尺度信息的深度融合。
  1. 双轨任务解码头

为适配 BCI 的多种解码范式，模型通过 UniBCIWrapper 在预训练骨架之上配置了灵活的分支解码逻辑：

  • 分类解码：将骨干输出的四维特征 (B, P×S, W, embed_dim) 展平为一维向量，依次馈入 Linear(dim, 4×embed_dim) → ReLU → Linear(4×embed_dim, num_classes)，输出类别 Logits。

  • 回归解码：同样的展平特征经 Linear(dim, 4×embed_dim) → ReLU → Linear(4×embed_dim, num_outputs × P × W) 后重塑为 (B, P×W, num_outputs)，实现时间分布式序列回归，精准适配连续运动学坐标的逐帧预测。

二、调用示例​

分类任务​

import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.unibci(task="classification")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

回归任务​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="regression",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.unibci(task="regression")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

默认情况下不加载预训练权重。如需加载预训练权重，请使用：

train_result = model.train(
    loaded_data.train_dataset, loaded_data.val_dataset,
    pretrained_path="auto",
)

Notes​

  • UniBCI 的分类与回归任务共享完全相同的默认超参数配置。两者的核心差异在于优化器与损失函数：分类采用 Adam + CrossEntropyLoss，回归采用 AdamW + CosineAnnealingLR + SmoothL1Loss。
  • 回归任务中，weight_decay 虽然参数默认值为 0，但实际生效值通过 max(weight_decay, 1e-4) 强制约束，即即便传入 0 也会以 1e-4 执行权重衰减。
  • val_ratio 仅在未提供外部验证集（val_dataset）时生效，用于从训练集中切分验证集。
  • pretrained_path 支持 "auto" 以自动下载 POYO-1 预训练权重，设为 None 则随机初始化。

四、模型调试与评估​

评估指标

  • 分类：准确率（Accuracy）+ 混淆矩阵。
  • 回归：在逆归一化后的原始空间计算 MSE、MAE、R2。

优化策略

  • 分类：Adam + CrossEntropyLoss。
  • 回归：AdamW + CosineAnnealingLR + SmoothL1Loss（β=1.0），配合梯度裁剪（max_norm=1.0）防梯度爆炸。

超参调试

  • 通道容量：max_group_num × group_size（默认 256）为最大通道数。超限时选方差最大的通道子集；不足时零填充自动生效，可减小 max_group_num 降无效计算。
  • 时间感受野：patch_num × patch_size（默认 100）。高采样率（>1kHz）增大 time_size 覆盖更长行为窗口，低采样率等比缩小。
  • 跨 Patch 注意力：attn_window（默认 10，即全对全）。增大可捕获长程耦合，但计算量线性增长。
  • 线性注意力：Intra-patch 用线性注意力换 O(n) 复杂度，窗口 W=10 时精度损失可忽略。patch_size > 50 时需关注近似误差。
  • 文本编码：仅多数据集联合预训练时有效。单数据集微调开启无增益，徒增推理开销。

五、应用说明​

该 UniBCI 实现框架在设计上以统一的 Token 化表征和混合注意力机制换取跨任务、跨数据集的泛化能力：

(1) 通用神经信号解码：UniBCI 是目前框架内唯一同时共享分类与回归预训练骨架的模型。其统一的脉冲 Token 化流程使得同一套参数可以在运动解码、感觉解码、认知状态识别等多种 BCI 范式之间无缝迁移，无需针对每种任务重新设计输入层。

(2) 大规模预训练与微调：得益于 POYO-1 的预训练权重初始化，UniBCI 在样本量较小的数据集上可通过少样本微调（few-shot fine-tuning）快速收敛。文本条件编码机制进一步支持多源异构数据的联合预训练，为构建跨物种、跨脑区的通用神经解码基础模型提供了工程基础。
