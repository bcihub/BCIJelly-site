6_Transformer

一、模型简介​

本模型提供了一套专门针对神经生理信号（如 Spike、LFP 等）设计的 Transformer 序列解码架构。脑机接口数据通常具有复杂的时序依赖性和跨通道相关性，传统 RNN 模型在处理长序列时易面临梯度弥散问题。本模型突破了这一局限，其核心假设在于：利用自注意力机制可以直接建立任意时间步之间的全局依赖关系，从而更高效地捕获神经群体活动的动态表征。

从代码实现与网络拓扑来看，本 Transformer 架构由以下核心模块构成：

  1. 输入投影层

    原始输入通常为高维特征通道的神经信号。模型首先通过线性投影层将特征维度映射至统一的嵌入维度，确保后续注意力计算的特征空间一致性。

  2. 正弦位置编码

    针对连续运动轨迹解码等强时序依赖任务，模型引入了固定周期的正弦/余弦位置编码。此机制将绝对时序信息注入到自注意力机制中，使模型能够感知脉冲序列的发生顺序。

  3. 多头自注意力编码器

    核心骨干网络由多层 TransformerEncoderLayer 堆叠而成。通过多头机制，模型在不同的子空间中并行计算查询、键和值的缩放点积注意力。该机制彻底打破了时序距离的限制，使得模型能够以全局感受野直接建立跨越数百毫秒的神经动作电位依赖关系。

  4. 双轨任务解码头

    为适配泛用的 BCI 范式，模型在输出端设计了高度灵活的解码逻辑：

    • 全局池化解码： 对于离散的意图识别或整段信号分类，模型在时间维度上进行均值池化，随后连接线性分类头。
    • 逐帧时序解码： 当开启序列目标预测时，模型绕过池化层，直接对每个时间步的隐藏状态进行线性映射，输出包含时间维度的特征张量，精准适配连续运动学坐标（如二维空间坐标）的实时回归预测。

二、调用示例​

分类任务​

import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.transformer(task="classification")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

回归任务​

四、模型调试与评估​

任务隔离的优化器调度

  • 分类任务：采用 Adam 优化器 + 交叉熵损失。
  • 回归任务：采用 AdamW 优化器 + 余弦退火学习率 + 平滑 L1 损失（提升对异常值的鲁棒性与收敛平滑度）。

多维评估指标

  • 分类：除准确率与损失外，输出完整混淆矩阵以追踪类别偏置。
  • 回归：同步计算均方误差、平均绝对误差及决定系数 (R2)。

自动化日志与可视化

  • 实时追踪训练指标及学习率。
  • 支持一键生成双 Y 轴训练趋势曲线与批量任务的最优指标对比图。

五、应用说明​

依据工程实践，本 Transformer 架构已被设计为足以应对主流的经典侵入式与非侵入式 BCI 数据集，具备极高的领域迁移能力：

(1) 多模态神经解码全覆盖：不仅能通过序列预测精准完成连续运动学解码（如皮层 Spike 追踪），还能利用全局池化机制高效实现多分类的离散意图与状态识别。

(2) 离线全局分析与块级在线解码优势：不同于 RNN，其对全局上下文的依赖赋予了显著的长时程特征挖掘能力，在离线分析中表现卓越；针对实时任务，则采用“块级在线”范式，以短暂延迟换取极高的时空特征分辨率。
