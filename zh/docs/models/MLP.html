2_MLP

一、模型简介​

多层感知机（Multi-Layer Perceptron, MLP）是最经典的前馈神经网络架构，其反向传播训练算法由 Rumelhart、Hinton 和 Williams 于 1986 年系统性地提出并推广。在脑机接口（BCI）领域，MLP 被广泛用作神经信号解码的性能基准。其核心假设在于：将高维神经群体活动通过多层非线性变换映射到行为标签或运动学状态，尽管模型本身不包含任何时序建模结构，但凭借其简洁的拓扑与极高的训练效率，在大量场景下仍展现出令人意外的竞争力。

从代码实现与网络拓扑来看，本 MLP 架构由以下核心模块构成：

  1. 时空展平输入层

模型不对原始神经信号进行任何时序建模。输入张量 (batch, time, channels) 首先被直接展平为一维向量 flatten(time × channels)。这一设计完全抛弃了时间步之间的先后关系，将所有时空信息视为一个无序的特征集合，通过后续全连接层的全局连接权重来隐式捕获特征间的统计关联。

  1. 级联全连接隐藏层

展平后的特征向量依次穿过三层全连接隐藏层，维度按 [512, 256, 128] 逐级压缩。每一层线性变换后跟随非线性激活函数（如 ReLU 或 GELU），并插入 Dropout 正则化（分类任务 0.3，回归任务 0.35）。这种"宽而浅"的金字塔结构在保留足够表达容量的同时，有效抑制了对小样本神经数据集的过拟合倾向。

  1. 双轨任务解码头

为适配 BCI 的多种解码范式，模型在输出端配置了灵活的分支解码逻辑：

  • 分类解码：最后一层隐藏状态馈入线性分类层，输出类别 Logits，经 Softmax 得到离散行为意图的概率分布。

  • 回归解码：最后一层隐藏状态馈入线性回归层，逐样本输出二维运动学坐标，实现连续运动轨迹的点估计。

  1. Z-Score 输入归一化

模型强制采用训练集均值和标准差的 Z-Score 归一化策略，将各神经通道统一至零均值、单位方差的尺度下参与前向传播。这一预处理步骤对于消除跨通道量纲差异、加速全连接层的收敛至关重要。

三、调用示例​

classification​


import bcijelly as bj

loaded_data = bj.load_data(
    dataset_name="5_Jango",
    protocol="cross_day", task="classification",
    subject_index=0, split_ratio="7:1:2",
)
model = bcijelly.model.mlp(task="classification")
train_result = model.train(loaded_data.train_dataset, loaded_data.val_dataset)
test_result = model.test(loaded_data.test_dataset)

bj.summary(loaded_data, model, train_result, test_result)

四、模型调试与评估​

评估指标

  • 分类任务：计算整体准确率，并生成混淆矩阵以诊断类别不平衡或错判偏好。
  • 回归任务：在自动逆归一化后的原始物理空间内，计算均方误差（MSE）、平均绝对误差（MAE）及决定系数。

核心调试技巧

  • Dropout 调节：dropout 是 MLP 最主要的正则化手段。分类任务默认 0.3，回归任务默认 0.35。对于通道数少、样本量小的数据集，可适当增大 dropout 以抑制过拟合；反之可减小。
  • 隐藏层维度：三层全连接 [512, 256, 128] 是针对典型神经数据集（数百通道、数千时间步）的默认配置。输入特征数远小于 512 时，应等比缩小隐藏层维度以避免严重过参数化。
  • 学习率与早停：Adam 优化器配合 patience=15 的早停机制。MLP 收敛速度通常快于 Transformer 等深度模型，80 个 epoch 内即可达到最优。
  • Z-Score 归一化：输入归一化对全连接网络至关重要。若数据未经归一化直接送入 MLP，大尺度通道将主导梯度更新，导致模型对其他通道失聪。

五、应用说明​

该 MLP 实现框架在设计上以极简拓扑换取最大通用性：

(1) 性能基线基准：作为不含任何时序归纳偏置（无卷积、无循环、无注意力）的纯前馈模型，MLP 在所有 BCI 解码器中的复杂度下界。如果在某个数据集上，复杂模型（如 Transformer 或 LFADS）的性能无法显著超越 MLP，则意味着该数据集的信息主要编码在通道间相关性而非时序结构中，这为后续模型选型提供了关键诊断信号。

(2) 高吞吐批量推理：得益于全连接层在 GPU 上的极致并行效率，MLP 的推理吞吐在所有解码器中最高（单样本前向仅涉及三次矩阵乘法），天然适配需要对海量神经数据进行快速打标或离线预筛选的应用场景。
